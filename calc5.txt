# 0. БАЗА

Метрика:

[
g_{\mu\nu} = \eta_{\mu\nu} + \frac{1}{M_{Pl}} \tilde h_{\mu\nu}

# 1. СКАЛЯРНАЯ ЧАСТЬ МЕТРИКИ

Разложение (\xi_{\mu\nu}):

[
\xi_{\mu\nu} = \xi^{TT}*{\mu\nu} + \frac{1}{4}\eta*{\mu\nu}\xi + ...
]

Скалярная часть:

[
h_{\mu\nu}^{(s)} = \phi \eta_{\mu\nu} - \frac{1}{4}\eta_{\mu\nu}\xi

Подставляем:

[
\xi = \bar h + 4\phi

[

\phi - \frac{1}{4}(\bar h + 4\phi)
= -\frac{1}{4}\bar h

# НОВЫЙ РАЗДЕЛ: ПРОВЕРКА МЕХАНИЗМА ПИСАРЕЙ

# A. СТАТУС ВСЕХ ПУТЕЙ К γ > 1

| механизм | γ | статус |
|---|---|---|
| P_k = (ρℓ³)^k, k=3 | 3 | ✗ требует ρℓ³ ≥ 1, нефизично при ℓ ~ ℓ_Pl |
| det(G_ij) писарей | 1 | ✗ det(G)=1 в плоском → не меняет scaling |
| Φ_eq(ρ) нелинейность | 2 (при ρℓ³≫1) | ✗ нестабильность при ρℓ³≪1 |
| self-consistent ℓ_c~1/m | <1 | ✗ calc4, подтверждено |
| κ_eff(ρ) = κ_0 + λρ/M_Pl² | 2 (при ρ≫ρ*) | ★ не исключён |

---

# B. МЕХАНИЗМ κ(ρ): ВЫВОД

[
[S_i, S_j]_{\rm matter} = i\kappa_{\rm eff}(\rho)\,\varepsilon_{ijk}\,S_k
]

[
\kappa_{\rm eff}(\rho) = \kappa_0 + \frac{\lambda\rho}{M_{Pl}^2}
]

[
m^2 \sim \kappa_{\rm eff}^2 = \kappa_0^2 + \frac{2\kappa_0\lambda}{M_{Pl}^2}\rho + \frac{\lambda^2}{M_{Pl}^4}\rho^2
]

Режимы:

[
\rho \ll \rho_* \equiv \frac{\kappa_0 M_{Pl}^2}{\lambda}: \quad m^2 \approx \kappa_0^2 + \frac{2\kappa_0\lambda}{M_{Pl}^2}\rho \sim \rho^1
]

[
\rho \gg \rho_*: \quad m^2 \approx \frac{\lambda^2}{M_{Pl}^4}\rho^2 \quad \Rightarrow \gamma = 2
]

---

алгебры [S_i, S_j] в sl(2,R).
