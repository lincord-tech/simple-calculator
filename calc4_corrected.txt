# ПОЛНЫЙ РАСЧЁТ ТЕОРИИ (С КРИТИЧЕСКИМИ ТОЧКАМИ)

---

# 0. БАЗА

Метрика:

[
g_{\mu\nu} = \eta_{\mu\nu} + \frac{1}{M_{Pl}} \tilde h_{\mu\nu}
]

Разложение:

[
h_{\mu\nu} = \phi,\eta_{\mu\nu} - \xi_{\mu\nu}
]

След:

[
\bar h = \xi - 4\phi
]

---

# 1. СКАЛЯРНАЯ ЧАСТЬ МЕТРИКИ

Разложение (\xi_{\mu\nu}):

[
\xi_{\mu\nu} = \xi^{TT}*{\mu\nu} + \frac{1}{4}\eta*{\mu\nu}\xi + ...
]

Скалярная часть:

[
h_{\mu\nu}^{(s)} = \phi \eta_{\mu\nu} - \frac{1}{4}\eta_{\mu\nu}\xi
]

Подставляем:

[
\xi = \bar h + 4\phi
]

[
h_{\mu\nu}^{(s)} =
\phi - \frac{1}{4}(\bar h + 4\phi)
= -\frac{1}{4}\bar h
]

---

# РЕЗУЛЬТАТ 1

[
\boxed{h_{\mu\nu}^{(s)} = -\frac{1}{4}\bar h,\eta_{\mu\nu}}
]

[
\phi \text{ исчезает полностью}
]

---

# 2. СВЯЗЬ С ФИЗИЧЕСКИМ ПОЛЕМ

Диагонализация:

[
\Phi = \phi + \frac{c}{M_{Pl}}\bar h
]

[

]

[
\Rightarrow \Phi = \phi + \frac{4}{M_{Pl}^3}\bar h
]

# ОЦЕНКА ПОРЯДКА

[

\Rightarrow \frac{4}{M_{Pl}^3}\bar h \sim \frac{1}{M_{Pl}^4}

---

# РЕЗУЛЬТАТ 2

]

---

# 3. МЕТРИКА

# 1. ДИСКРЕТНАЯ СРЕДА

[

[

# 2. СЛУЧАЙНЫЕ СТОЛКНОВЕНИЯ

пусть взаимодействие требует **k частиц одновременно**

вероятность:

[
P_k = \text{Prob}(\text{в ячейке ≥ k частиц})
]

для пуассоновской среды:

[
P_k \sim \frac{N^k}{k!}
\quad (N \ll 1 \text{ не обязателен, но scaling сохраняется})
]

---

# 3. АМПЛИТУДА ПЕРЕХОДА

[

\left[1 - \lambda P_k(x),\Delta t \right]
\Phi(x+\Delta x_i)
]

# ⚠ ЗАМЕЧАНИЕ: ad hoc

Ядро K(t) ~ δ'(t) выбрано специально чтобы убрать квадрат.
Это не вывод из микроскопики — это постулирование нужного ответа.

Физическое оправдание: нелокальность по времени
возникает если поле взаимодействует не мгновенно,
а через «память» среды (ретардированный отклик).

Для самосогласованности нужно вывести K из
пропагатора флуктуаций поля (см. [D8] из Part II).
Без этого шаг 13→14 — предположение, не вывод.

# МИКРОМОДЕЛЬ БЕЗ FINE-TUNING

---

# 1. СРЕДА С КОРРЕЛЯЦИЯМИ

Пусть плотность имеет флуктуации:

[
\rho(x) = \bar\rho + \delta\rho(x)
]

[
\langle \delta\rho(x) \delta\rho(0) \rangle = C(r)
]

---

# 2. КОРРЕЛЯЦИОННЫЙ ОБЪЁМ

Определим:

[
\ell_c^3 = \int d^3r , C(r)
]

эффективное число частиц:

[
N_{eff} = \rho \ell_c^3
]

---

# 3. КОЛЛЕКТИВНОЕ ВЗАИМОДЕЙСТВИЕ

пусть поле взаимодействует с кластером:

[
S_{int} = -\lambda \int d^4x , \Phi^2 , \mathcal{O}(x)
]

[
\mathcal{O}(x) = \sum_{i=1}^{N_{eff}} 1
]

[
\Rightarrow \mathcal{O} \sim N_{eff}
]

---

# 4. ФЛУКТУАЦИИ КЛАСТЕРОВ

распределение:

[
P(N) \sim e^{-N/N_*}
]

среднее:

[
\langle N^k \rangle \sim N_*^k
]

---

# 5. ЭФФЕКТИВНЫЙ ЧЛЕН

интегрируя по кластерам:

[
S_{eff} \sim -\lambda \langle N \rangle \Phi^2
]

но с корреляциями:

[
\langle N \rangle \to \langle N^k \rangle^{1/k}
]

---

# 6. МАССА

[
m^2 \sim \lambda N_*^k
]

[
N_* \sim \rho \ell_c^3
]

---

[
m^2 \sim \lambda (\rho \ell_c^3)^k
]

---

корреляционная длина из флуктуаций:

[
\ell_c^2 \sim \frac{1}{m^2}
]

---

# 8. ЗАМЫКАНИЕ

[

]

---

# 9. РЕШЕНИЕ

# ⚠ ЗАМЕЧАНИЕ: одно исключение

Вывод γ < 1 опирается на замыкание:

[
\ell_c^2 \sim \frac{1}{m^2}
]

Это перестаёт работать если среда находится
вблизи фазового перехода (перколяция, критическая точка):

[
\ell_c \to \infty \quad (\text{независимо от } m)
]

В этом случае замыкание ломается и γ может быть > 1.

Это **единственный физически мотивированный** способ
получить нужный scaling без fine-tuning.

Требует: отдельного анализа с потенциалом V(ρ),
допускающим фазовый переход.

---

# ИТОГ

без fine-tuning и вне критической точки:

[
\boxed{m^2 \not\sim \rho^k,\; k>1}
]

---

# ФИНАЛ

единственный способ получить:

[
\gamma > 1
]

это:

* жёстко заданный многочастичный процесс (k ≥ 3)
* или внешняя шкала (критическая точка / фазовый переход)
* или нелокальное взаимодействие (D8 из Part II)

---

# ЗАКЛЮЧЕНИЕ

[
\boxed{\text{без fine-tuning теория не даёт нужного scaling}}
]

---

# ИТОГОВАЯ ОЦЕНКА

Теория **не мертва автоматически**, но находится под давлением.

✔ Математика секций 0–10 (метрика, coupling, действие) — верна
✔ Screening на планетарных масштабах — работает
✔ Отсутствие double counting — подтверждено (секция 7–8)

❌ Лабораторный тест не проходит при γ = 1 (k = 1)
❌ Получить γ ≥ 3 естественно — не удаётся без дополнительной физики

Реальная точка убийства:
не lattice, не coupling, не нормировка —

[
\boxed{
\text{невозможность получить } \gamma \ge 3 \text{ без внешней шкалы}
}
]

Дальше два честных пути:

1. Искать физику, где появляется внешняя шкала
   (фазовый переход / перколяция / нелокальность)

2. Признать: m² ~ ρ (γ = 1) →
   теория экспериментально убита в текущем виде
