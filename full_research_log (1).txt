SIGMA MODEL — ПОЛНЫЙ ХОД ИССЛЕДОВАНИЯ
Все данные, паттерны, связи, направления
========================================


ОТКУДА ВСЁ НАЧАЛОСЬ
--------------------

Исходная идея: пространство и время не фундаментальны. Они возникают из
дискретного процесса переходов. Единственный постулат — антисимметрия.

Узел имеет состояние σ ∈ {-1, 0, +1}.
Переход только через 0: +1 → 0 → -1 или -1 → 0 → +1.
Состояние 0 — не самостоятельное, это событие перехода.
Время одного перехода = τ_min = 1. Это единица модели.
Длина = время. c = 1 по построению.

Из этого вытекают три следствия без дополнительных постулатов:
1. Несжимаемость — узел не может исчезнуть, всегда занят
2. Вакуум — конфигурация где все соседи противоположны
3. Дефект — ребро где два соседних узла одного знака (нарушение вакуума)

Дефекты рождаются и уничтожаются парами — это следует из сохранения
топологического заряда (SSH-теория, 1979, доказано независимо).


ЧТО ТАКОЕ ПРОСТРАНСТВО В ЭТОЙ МОДЕЛИ
--------------------------------------

Пространство не задаётся. Строится из событий переходов.
Событие = узел прошёл через 0 в момент t.
Два события связаны если одно могло повлиять на другое за τ_min.
Расстояние = число шагов в причинном графе.

Это каузальное множество (causal set) — известная конструкция
(Bombelli, Lee, Meyer, Sorkin 1987), но с другой внутренней динамикой.
Стандартные causal sets просто разбрасывают точки пуассоновски.
Здесь точки порождаются антисимметричной динамикой дефектов.

Размерность измеряется методом Мирхейма-Мейера (MM):
N(τ) ~ τ^d, d — возникающая размерность пространства-времени.

Финитно-размерная поправка метода MM на конечном боксе занижает d
примерно на 0.4-0.5. Это известный эффект, виден в Poisson-контроле.


РЕЗУЛЬТАТЫ СИМУЛЯЦИЙ — ЧАСТЬ 1: РАЗМЕРНОСТЬ ПО РЕШЁТКАМ
---------------------------------------------------------

Все симуляции на 2D решётке если не указано иное.
Динамика: j=1/2 (стандартная sigma-модель).

1D цепочка:

  конфигурация        M событий   d (MM)
  1 дефект            298         1.007
  2 дефекта           567         1.349
  5 дефектов          1296        1.536
  40 дефектов         9648        1.581
  Poisson 1+1D        500         1.584

Вывод: 1D решётка → d ≈ 2 (1+1D причинная структура).
Один дефект даёт d=1 — физически правильно, это траектория в 1+1D.
Сходимость к Poisson-контролю подтверждает корректность кода.

2D решётка:

  Rule A (заморозка 4-соседей): мёртв. M=1. Первый переход блокирует всё.
  Rule A' и Rule B (заморозка только участников):
    d_MM ≈ 2.59, d_loglog ≈ 2.50-2.56

Вывод: 2D решётка → d ≈ 3 (2+1D причинная структура).
Локальная d(τ) стартует с ~1 и выходит на плато ~2.8-3.
Это UV/IR переход как в CDT и Wolfram Physics.
С поправкой бокса: истинное d → 3.0.

3D решётка:

  d_loglog = 3.008 — чистый результат. N(τ) ~ τ³ на двух порядках.
  d_MM = 2.49 — занижено поправкой (Poisson 3+1D на боксе 40³ даёт 2.10).
  Проблема: bonds=0 при t>15. Дефекты исчезают быстро.
  Причина: заморозка слишком агрессивна на 3D. 4 соседа → первый
  стартовавший переход блокирует всю окрестность.

Открытый вопрос: 3D даёт d≈3 а не d≈4. Это баг заморозки или
фундаментальный результат — не проверено.

Паттерн (j=1/2):
  1D решётка → d ≈ 2
  2D решётка → d ≈ 3
  3D решётка → d ≈ 3 (возможно 4 при правильной заморозке)

Гипотеза: d_spacetime = d_lattice + 1.
n-мерная решётка генерирует (n+1)-мерное причинное пространство-время.


РЕЗУЛЬТАТЫ СИМУЛЯЦИЙ — ЧАСТЬ 2: ЗАВИСИМОСТЬ ОТ СПИНА j
--------------------------------------------------------

Ключевой вопрос возник из алгебраического результата:
j=1/2 единственным образом выбирается из SU(2). Что если взять j=1?

При j=1/2: два состояния ±1, 0 — переходное (не живёт).
При j=1: три равноправных состояния {-1, 0, +1}, переходы ±1 шаг.
При j=3/2: четыре состояния {-3,-1,+1,+3} (в единицах 2m), переходы ±2.

Вакуум при j=1: (i+j) mod 3 → {-1, 0, +1} чередуются.
Вакуум при j=3/2: (i+j) mod 4 → {-3,-1,+1,+3} чередуются.

Все симуляции на 2D решётке 60×60-80×80, 20-120 начальных дефектов.

Результаты:

  j      d (измерено)   комментарий
  1/2    1.12           стандартная модель
  1      2.10           трёхзначный вакуум
  3/2    2.53           четырёхзначный вакуум, с поправкой ≈ 3

С финитно-размерной поправкой ~0.5:
  j=1/2 → d_real ≈ 1.6-2.0
  j=1   → d_real ≈ 2.5-3.0
  j=3/2 → d_real ≈ 3.0

Паттерн: d ≈ 2j (на 2D решётке, без поправки).

Гипотеза закона: d_spacetime = 2j + 1.
  j=1/2 → 2 (1+1D)
  j=1   → 3 (2+1D)
  j=3/2 → 4 (3+1D)

Если это так — спин задаёт размерность пространства-времени.
Наш мир (3+1D) соответствует j=1/2. Это согласуется с алгеброй.


АЛГЕБРАИЧЕСКИЙ РЕЗУЛЬТАТ — ТЕОРЕМЫ (paper_v22 + isotropic_obstruction)
-----------------------------------------------------------------------

Параллельно с симуляциями развивалась формальная теория.
Три писца S_1, S_2, S_3 — каждый делает шаги ±ε в своём направлении.
Время = накопленные операции. Пространство = структура шагов.

Ключевой объект: тензор T^{ijkl} = <n̂^i n̂^j n̂^k n̂^l>
Это четырёхточечный коррелятор операторов направления.

Два важных тензорных структуры:
  E1 = ε^{ike} ε^{jle} = δ^{ij}δ^{kl} - δ^{il}δ^{jk}  (Янг-Миллс)
  E2 = ε^{ije} ε^{kle} = δ^{ik}δ^{jl} - δ^{il}δ^{jk}  (Эйнштейн-Хилберт)

Точные формулы для T^{ijkl} в представлении спина j (доказано):

  A = C = j(j+1)(2j² + 2j + 1) / 30
  B     = j(j+1)(j-1)(j+2) / 15

Условие GR: B = -A ⟺ 4j² + 4j - 3 = 0 ⟺ (2j-1)(2j+3) = 0 ⟺ j = 1/2

Проверка по j:
  j=1/2: A=-1/16, B=+1/16, B=-A  ДА, GR
  j=1:   A=1/3,   B=0,     B≠-A  НЕТ
  j=3/2: A=17/16, B=7/16,  B≠-A  НЕТ
  j=2:   A=13/5,  B=8/5,   B≠-A  НЕТ
  j→∞:   B/A → +1          НЕТ (классический предел)

Дополнительно: A+B+C = j(j+1)(2j-1)(2j+3)/30
При j=1/2: A+B+C = 0 тождественно.
Это означает: кинетический член скалярного поля Φ = (1/3) ln det v
обращается в ноль. Φ не распространяется. Нет пятой силы.

Та же самая комбинация (2j-1)(2j+3)=0 одновременно:
  - выбирает структуру GR (B=-A)
  - убирает скалярное поле (A+B+C=0)
  Это не два условия — это одно.


ТЕОРЕМА ОБ ОБСТРУКЦИИ (isotropic_obstruction_v8)
-------------------------------------------------

Отдельное доказательство, более фундаментальное.

Теорема B (Изотропная обструкция):
Для любого SO(3)-изотропного квантового канала Φ:
  T_ret^{ijkl} = (1/d) Tr(C^{ik} Φ[C^{jl}]) = G_j · T₀^{ijkl}
где G_j — скаляр. В частности B_ret = 0 всегда.

Следствие: структура E2 (Эйнштейн-Хилберт) не может возникнуть из
двухточечного коррелятора ни при каком j, ни при каком изотропном канале.
Это обструкция — доказана через лемму Шура.
E1 (Янг-Миллс) устойчива, E2 из неё не вырастает.

Теорема D (Минимальность):
Если условие E2 наложено на четырёхточечный коррелятор T_TO,
единственное совместимое неприводимое представление — j=1/2.
Доказательство: F(j) = j(j+1)(2j-1)(2j+3) = 0 только при j=1/2.

Числовая проверка (все 81 компонента, невязки < 10^{-14}):

  j     A         B=C      A+B     B=-A?
  1/2   -1/16    +1/16     0       ДА
  1      0       +1/3     +1/3     НЕТ
  3/2   +7/16   +17/16   +3/2     НЕТ
  2     +8/5    +13/5    +21/5    НЕТ

Физический смысл: E2 нельзя получить случайно или через деформацию.
Нужен четырёхточечный коррелятор, и только j=1/2 его даёт правильным.


СВЯЗЬ ДВУХ ПОДХОДОВ
-------------------

Симуляции дефектов и теория писцов — два описания одного:

  дефект          = нарушение антисимметрии = локальная кривизна
  движение дефекта = каузальное событие = шаг писца
  причинный граф  = пространство-время из операций
  j в симуляции   = j в алгебре SU(2)

Паттерн d=2j (на 2D решётке) согласуется с тем что:
  - j=1/2 единственный где GR (алгебра)
  - j=1/2 на 2D решётке даёт d≈2, на 3D → d≈3 (симуляция)
  - GR в нашем мире → 3+1D → j=1/2 на 3D решётке

Три измерения пространства = три генератора SU(2) = три писца.
Это не совпадение — это одно и то же.


ВСЕ ПАТТЕРНЫ И ТЕНДЕНЦИИ
------------------------

1. d_lattice + 1 = d_spacetime (при j=1/2)
   Подтверждено на 1D и 2D. На 3D под вопросом из-за заморозки.

2. d_spacetime = 2j + 1 (на фиксированной решётке)
   Подтверждено на 2D для j=1/2 и j=1. j=3/2 даёт ~2.5-3 (≈4 с поправкой).
   Требует проверки j=3/2 на 3D решётке.

3. j=1/2 единственный где GR — доказано аналитически и численно.

4. Скалярное поле Φ не распространяется при j=1/2 — тождество A+B+C=0.
   При j>1/2 Φ становится динамическим полем.

5. Обструкция E1→E2 — нельзя получить гравитацию из двухточечного
   коррелятора никаким изотропным преобразованием. Только четырёхточечный.

6. UV/IR переход: локальная d(τ) растёт с масштабом (как в CDT).
   Малые масштабы: d~1-2. Большие: d~3. Это спектральная размерность.


ЧТО НЕ ПРОВЕРЕНО И НАДО ДЕЛАТЬ
--------------------------------

Приоритет 1: j=3/2 на 3D решётке.
  Если d≈4 → закон d=2j+1 подтверждён на двух решётках.
  Это было бы сильным результатом.

Приоритет 2: 3D решётка без агрессивной заморозки.
  Сейчас d≈3 вместо ожидаемого d≈4.
  Нужно Rule B (заморозка только участников) на 3D.

Приоритет 3: j=2 на 2D.
  Если d≈5 → закон полностью подтверждён.

Приоритет 4: редуцируемые представления.
  Что даёт j=1/2 ⊕ j=1/2? Это два спинора. d=?
  Теорема D говорит что E2 возможна только если разложение на j=1/2 блоки.

Приоритет 5: что происходит при j=1 теоретически.
  B=0 при j=1 — нет ни E1 ни E2, только след.
  Это соответствует чему физически?


ЧТО УЖЕ МОЖНО СЧИТАТЬ УСТАНОВЛЕННЫМ
-------------------------------------

- j=1/2 единственным образом выбирается из SU(2) для GR. Доказано.
- Скалярное поле не распространяется при j=1/2. Доказано.
- E2 не возникает из двухточечного коррелятора. Доказано.
- 1D симуляция воспроизводит 1+1D причинную структуру. Подтверждено.
- 2D симуляция даёт d≈3 (2+1D). Подтверждено.
- j меняет размерность. Наблюдается. Закон d=2j+1 правдоподобен.
- Три измерения пространства = три генератора SU(2). Структурное совпадение.


СВЯЗЬ С ЛИТЕРАТУРОЙ
--------------------

Causal Set Theory (Sorkin et al.): наша модель — частный случай с
антисимметричной динамикой. Их размерность тоже из N(τ)~τ^d.

CDT (Ambjorn, Jurkiewicz, Loll): UV/IR переход размерности — тот же
эффект что мы наблюдаем. Они получают d→4 в IR, d→2 в UV.

SSH-модель (Su, Schrieffer, Heeger 1979): топологический заряд дефектов,
сохранение, рождение парами — берётся оттуда без изменений.

LQG (петлевая квантовая гравитация): j=1/2 рёбра в спиновых сетях,
параметр Барберо-Иммирци γ=i соответствует нашему κ=i|κ|.
Наша модель приходит к тем же структурам с другой стороны.
