An Obstruction to Einstein-Hilbert Structure
in Isotropic SU(2) Correlators,
and a Minimality Theorem for j = 1/2


ABSTRACT

Let J^1, J^2, J^3 be the standard angular momentum operators in an irreducible
spin-j representation of SU(2) of dimension d = 2j+1. Define C^{ab} := -|kappa|
epsilon^{abk} J^k (kappa purely imaginary) and the rank-4 two-point tensor
T_0^{ijkl} = (1/d) Tr(C^{ik} C^{jl}). We prove three results:

(i) In the maximally mixed state rho = I/d, the tensor T_0 is purely E1
(Yang-Mills) with coefficient |kappa|^4 j(j+1)/3 > 0.

(ii) For any SO(3)-isotropic quantum channel Phi, the deformed correlator
(1/d) Tr(C^{ik} Phi[C^{jl}]) is proportional to T_0; in particular it always
has E1 structure. This constitutes an obstruction: no SO(3)-isotropic
deformation can produce E2 (Einstein-Hilbert) structure. The proof uses
Schur's lemma applied to the adjoint action of SU(2).

(iii) If E2 structure is imposed on the four-point correlator
T_TO^{ijkl} = (1/d) Tr(J^i J^k J^j J^l), the unique compatible irreducible
representation is j = 1/2, dim H = 2.

All results are verified numerically for j = 1/2, 1, 3/2, 2.


1. INTRODUCTION

The emergence of geometric tensor structures from algebraic data is a central
theme in approaches to quantum gravity. A concrete version of this question
arises in the context of SU(2) representation theory: given the generators
J^1, J^2, J^3 of an irreducible spin-j representation, two natural rank-4
tensors arise from products of generators, and they exhibit qualitatively
different tensor structures corresponding to Yang-Mills (E1) and
Einstein-Hilbert (E2) kinetic terms in gauge and gravitational theories.

The two-point tensor, constructed from C^{ab} := -|kappa| epsilon^{abk} J^k, is

  T_0^{ijkl} = (1/d) Tr(C^{ik} C^{jl}),

and the four-point tensor is

  T_TO^{ijkl} = (1/d) Tr(J^i J^k J^j J^l).

Both control kinetic structures of the form integral T^{ijkl} F^{ij} F^{kl}.
The two SO(3)-isotropic epsilon-structures relevant to gauge theory and gravity
are

  E1^{ijkl} = epsilon^{ike} epsilon^{jle} = delta^{ij} delta^{kl} - delta^{il} delta^{jk}   (Yang-Mills),
  E2^{ijkl} = epsilon^{ije} epsilon^{kle} = delta^{ik} delta^{jl} - delta^{il} delta^{jk}   (Einstein-Hilbert).

Any SO(3)-isotropic rank-4 tensor decomposes uniquely as

  T^{ijkl} = A delta^{ij} delta^{kl} + B delta^{ik} delta^{jl} + C delta^{il} delta^{jk}.   (DEC)

In this decomposition, E1 corresponds to (A=1, B=0, C=-1) and E2 to
(A=0, B=1, C=-1). The E2 condition is therefore B = -A.

This paper answers two questions: (a) Which structure does T_0 have, and can
any isotropic deformation change it? (b) For which representations does T_TO
satisfy the E2 condition? The answers: (a) T_0 is always E1 and no isotropic
quantum channel can change this (Theorem B); (b) only j = 1/2 (Theorem D).
Together these results sharply characterise the boundary between Yang-Mills and
Einstein-Hilbert kinetic structures in SU(2) representation theory.


2. ALGEBRAIC SETUP

Fix a spin-j irreducible representation of SU(2), with d = 2j+1. The generators
satisfy the commutation relations

  [J^p, J^q] = i epsilon^{pqr} J^r,

and the normalisation

  (1/d) Tr(J^p J^q) = j(j+1)/3 * delta^{pq}.   (*)

Fix kappa purely imaginary and set C^{ab} := -|kappa| epsilon^{abk} J^k.
Any SO(3)-isotropic rank-4 tensor has the unique decomposition (DEC) above.


3. THE TWO-POINT CORRELATOR T_0

Proposition. In the maximally mixed state rho = I/d,

  T_0^{ijkl} = |kappa|^4 * j(j+1)/3 * E1^{ijkl}.

Explicitly: A = |kappa|^4 j(j+1)/3, B = 0, C = -A.

Proof. Expand C^{ik} C^{jl} = |kappa|^2 epsilon^{ike} epsilon^{jlf} J^e J^f
and apply (*):

  T_0^{ijkl} = |kappa|^4 j(j+1)/3 * epsilon^{ike} epsilon^{jle}
             = |kappa|^4 j(j+1)/3 * E1^{ijkl}.

The overall coefficient is positive because |kappa|^4 > 0 (the two minus signs
from C^{ab} yield a positive factor). Since B = 0 in (DEC), T_0 is purely E1.
[]


4. MAIN THEOREM: ISOTROPIC OBSTRUCTION

A quantum channel Phi is SO(3)-isotropic if Phi[U O U^+] = U Phi[O] U^+ for
all U in SU(2). This class includes the identity channel, unitary conjugations,
depolarising channels, and all convex combinations thereof.

Theorem B (Isotropic Obstruction). For any SO(3)-isotropic Phi,

  T_ret^{ijkl} := (1/d) Tr(C^{ik} Phi[C^{jl}]) = G_j * T_0^{ijkl}

for a scalar G_j depending only on j and Phi. In particular B_ret = 0: the
deformed correlator always has E1 structure.

(We use the label T_ret as shorthand for this channel-deformed two-point tensor;
it is not a retarded Green's function in the standard field-theoretic sense.)

Proof. Let W = span{J^1, J^2, J^3} in End(H) equipped with the Hilbert-Schmidt
inner product <A, B> = (1/d) Tr(A^dag B). Under conjugation, W carries the
adjoint (spin-1) irreducible representation of SU(2). Since Phi commutes with
this SU(2) action, Schur's lemma gives Phi|_W = G_j * id_W, so

  Phi[J^k] = G_j J^k + X^k,    <J^a, X^k> = 0 for all a.

Since C^{ik} = -|kappa| epsilon^{ike} J^e lies entirely in W,

  (1/d) Tr(C^{ik} Phi[C^{jl}]) = G_j (1/d) Tr(C^{ik} C^{jl}) = G_j T_0^{ijkl}.   []

Corollary. No correlator (1/d) Tr(C^{ik} Phi[C^{jl}]) with SO(3)-isotropic Phi
has E2 structure.


5. MINIMALITY OF j = 1/2

Theorem B shows that the two-point tensor T_0 is irreparably E1. We now ask:
for which representations does the four-point correlator T_TO^{ijkl} =
(1/d) Tr(J^i J^k J^j J^l) satisfy the E2 condition B = -A? The answer is:

Theorem D (Minimality). Let H carry an SU(2) action via Hermitian generators
J^a satisfying the commutation relations, and define T_TO as above with the
E2 condition (3): T_TO has E2 structure, i.e., B = -A in (DEC).
Then dim H = 2 and j = 1/2.

Proof. Define F(j) = j(j+1)(2j-1)(2j+3). The derivation of F from condition
(3) is carried out in full in Appendix C; the result is

  A + B = F(j)/30.

Case dim H = 1: [J^a, J^b] = 0, contradicting the commutation relations.

Case H irreducible, spin j: The E2 condition requires A + B = 0, hence F(j) = 0.
Since j(j+1) > 0 for all j > 0, this reduces to (2j-1)(2j+3) = 0, giving
j = 1/2. The root j = -3/2 is excluded: it does not correspond to a unitary
representation (the inner product fails to be positive-definite). For j >= 1
all factors of F(j) are positive.

Case H reducible: See Appendix B. No reducible representation satisfies
condition (3) unless it decomposes into j = 1/2 blocks, and the minimal
irreducible subspace satisfying all conditions is V_{1/2} itself.

Therefore j = 1/2 is the unique solution, dim H = 2.   []

Remark. Theorem D does not assert that E2 structure arises from the SU(2)
algebra -- Theorem B shows it does not. The result is: imposing E2 on T_TO is
consistent with the commutation relations only at j = 1/2. Put differently,
only the qubit makes the four-point angular-momentum correlator exhibit
Einstein-Hilbert tensor structure.


5A. E2-NULLIFICATION: STRUCTURAL INTERPRETATION

The result of Theorem D admits a structural reformulation that makes its
algebraic content explicit and suggests natural generalisations.

Key object. Consider the symmetric part of the product of generators:

  (1/2){J^a, J^b} = j(j+1)/3 * delta^{ab} * I + Q^{ab},
  Q^{ab} := Pi_2(J^a J^b),

where Pi_2 denotes the projection onto the spin-2 (ell=2) irreducible component.
The operator Q^{ab} is the quadrupole operator: symmetric, traceless,
transforming in the spin-2 representation of SO(3).

Decomposition of Sym^2(rho_j). For general j:

  Sym^2(rho_j) = rho_0 + rho_2 + rho_4 + ... + rho_{2j}   (even spins only).

Q^{ab} corresponds exclusively to the rho_2 component. Higher even spins
rho_4, ..., rho_{2j} do not enter the definition of Q^{ab}.

Quadrupole norm. Define:

  lambda_j = C_j * (1/d) Tr(Q^{ab} Q^{ab}),   C_j > 0.

Since <.,.> > 0 and C_j > 0:

  lambda_j = 0  iff  Q^{ab} = 0.

Connection to multiplicity. The multiplicity of rho_2 in Sym^2(rho_j) is:

  m_2(rho_j) := dim Pi_2(Sym^2(rho_j)).

Since the Lie algebra generated by J^a acts irreducibly on H, the polynomial
algebra generated by rho(J^a) is isomorphic to End(H). Therefore the projection
Pi_2(J^a J^b) is nontrivial if and only if rho_2 appears in Sym^2(rho_j):

  Q^{ab} = 0  iff  m_2(rho_j) = 0.

This excludes the potential objection that rho_2 might be present in Sym^2(rho_j)
without being excited by Q^{ab}.

Theorem (E2-Nullification, SU(2)). Let Q^{ab} := Pi_2(J^a J^b). The following
conditions are equivalent:

  (1) E2-structure is realised (i.e., B = -A in (DEC))
  (2) Q^{ab} = 0
  (3) (1/d) Tr(Q^{ab} Q^{ab}) = 0
  (4) m_2(rho_j) = 0

The chain of equivalences:

  (1) <--(Appendix C)--> (3) <--(C_j>0, <.,.>>0)--> (2) <--(End(H))--> (4).

Corollary (SU(2)):

  E2 + <.,.> > 0  iff  j = 1/2.

At j = 1/2: Sym^2(rho_{1/2}) = rho_1, the rho_2 component is absent
structurally, Q^{ab} = 0 identically.

At j >= 1: rho_2 is contained in Sym^2(rho_j), m_2 > 0, Q^{ab} != 0.

Structural meaning. The E2-condition is determined not by the value of j but
by the absence of the quadrupole component in the operator algebra:

  E2  iff  Pi_2(J^a J^b) = 0.

This is a structural criterion -- it does not depend on any specific numerical
computation and transfers directly to any algebra with a distinguished tensor
sector of rank 2.

Two layers. The exclusion of the formal root j = -3/2 operates at a prior
level: no unitary representation with those quantum numbers exists, so the
positive-definite inner product required by condition (3) is not available.
The E2-Nullification theorem applies only within the class of unitary
representations.

  Layer I  (representational):  rho unitary  -->  excludes j = -3/2
  Layer II (algebraic):         m_2(rho_j) = 0  -->  selects j = 1/2

7. CONCLUSION

Theorem B establishes that the E1 (Yang-Mills) structure of T_0 is stable
under all SO(3)-isotropic deformations. More precisely, it constitutes an
obstruction: the E2 (Einstein-Hilbert) tensor structure cannot arise from any
isotropic quantum channel applied to the two-point correlator, regardless of
the representation j. The proof is exact and relies only on Schur's lemma and
the invariance of the adjoint representation under isotropic channels.

Theorem D establishes that if E2 structure is imposed on the four-point
correlator T_TO, the only compatible irreducible representation is j = 1/2.
The algebraic reason is the unique zero of F(j) = j(j+1)(2j-1)(2j+3) at
j = 1/2, together with positivity of the inner product which excludes the
formal root j = -3/2.

Section 5A reformulates Theorem D as the E2-Nullification theorem: the E2
condition is equivalent to the vanishing of the quadrupole operator Q^{ab} =
Pi_2(J^a J^b), or equivalently to the absence of the spin-2 component in
Sym^2(rho_j). The chain of equivalences

  E2  <-->  F(j) = 0  <-->  lambda_j = 0  <-->  Q^{ab} = 0  <-->  m_2(rho_j) = 0

shows that the E2 condition reflects not a numerical coincidence but a
structural property of the operator algebra: the qubit is the unique unitary
SU(2) representation in which the quadrupole sector is absent.

Together, these results sharply characterise the boundary between Yang-Mills
and Einstein-Hilbert kinetic structures in terms of SU(2) representation
theory: the E1 structure is universal and obstruction-stable, while the E2
structure is fragile and representation-selective. Which mechanism, if any,
selects E2 over E1 in a physical context -- and whether an analogous
obstruction persists for compact groups beyond SU(2) -- remain open questions.


APPENDIX A: EXPLICIT CHANNEL FOR TABLE 2

The isotropic channel used in Table 2 is the average over six elementary
SU(2) rotations by angle 1:

  Phi[O] = (1/6) sum_a [ e^{-iJ^a} O e^{+iJ^a} + e^{+iJ^a} O e^{-iJ^a} ].

Isotropy is manifest by construction: conjugating by any U in SU(2) commutes
with the average over rotations, since the set of rotation axes is SO(3)-invariant.

Verification for j = 1/2. With J^a = sigma^a/2, one computes explicitly:
Phi[J^k] = G_{1/2} J^k exactly.

Verification for j = 1, 3/2. One computes Phi[J^k] = G_j J^k + X^k and
verifies numerically that (1/d) Tr(J^p (X^k)^dag) = 0 to within 10^{-14},
confirming the trace-orthogonality <J^p, X^k> = 0 used in the proof of
Theorem B.


APPENDIX B: REDUCIBLE CASE IN THEOREM D

We show that the E2 condition is incompatible with any reducible decomposition
H = V_{j_1} + V_{j_2} with j_1 <= j_2, j_1, j_2 > 0.

In the block decomposition, the four-point correlator is weighted by the
relative dimensions. Let d_a = 2j_a+1 and lambda = d_1/(d_1+d_2). Then

  T_TO = lambda * T_TO(j_1) + (1-lambda) * T_TO(j_2).

The E2 condition A + B = 0 then requires

  lambda * F(j_1)/30 + (1-lambda) * F(j_2)/30 = 0,

i.e., lambda F(j_1) + (1-lambda) F(j_2) = 0.

We analyse the sign of F(j) = j(j+1)(2j-1)(2j+3) for half-integer j > 0:

  F(1/2) = 0  (unique zero for j > 0).
  F(j) > 0  for all j >= 1.

With lambda in (0,1), both terms are non-negative and their weighted sum
vanishes only if both vanish individually, requiring F(j_1) = F(j_2) = 0,
which forces j_1 = j_2 = 1/2.

In the case H = V_{1/2} + V_{1/2}, the correlator T_TO is the equal-weight
average of two identical j = 1/2 correlators and therefore itself satisfies
the E2 condition. However, the minimal irreducible subspace satisfying
conditions (1)-(3) is V_{1/2} itself. This completes the proof.   []


APPENDIX C: DERIVATION OF THE COEFFICIENTS OF T_TO

We derive, from first principles, the closed-form expressions for the
coefficients A, B, C in the isotropic decomposition (DEC) of the four-point
correlator

  T_TO^{ijkl} = (1/d) Tr(J^i J^k J^j J^l)

in an irreducible spin-j representation of SU(2), with d = 2j+1.


C.1 Operator Product Decomposition

Any product of two generators decomposes under the adjoint action of SU(2):

  J^a J^b = (i/2) epsilon^{abc} J^c + [j(j+1)/3] delta^{ab} I + Q^{ab},

where the antisymmetric part is fixed by the algebra, the second term is the
scalar (spin-0) part, and Q^{ab} is the quadrupole operator: symmetric,
traceless, transforming in the spin-2 representation of SO(3).


C.2 Symmetry of T_TO under Sym^2(rho_j)

The Lie algebra acts irreducibly on H, so the polynomial algebra generated by
rho(J^a) is isomorphic to End(H). The symmetric part (1/2){J^a, J^b} generates
Sym^2(rho_j), which decomposes under SO(3) into even-spin components only:

  Sym^2(rho_j) = rho_0 + rho_2 + rho_4 + ... + rho_{2j}.

The rho_2 component is precisely Q^{ab}. Since rho(J^a) generates End(H)
irreducibly, the presence or absence of rho_2 in Sym^2(rho_j) is equivalent
to Q^{ab} != 0 or Q^{ab} = 0 respectively. At j = 1/2 specifically, H is
2-dimensional and Sym^2(rho_{1/2}) = rho_1 contains only ell = 1, so
Q^{ab} = 0 identically.


C.3 The Quadrupole Norm

By Schur's lemma, the SO(3)-covariant bilinear pairing of Q^{ab} with itself
is proportional to the unique SO(3)-invariant projector onto symmetric
traceless rank-2 tensors:

  (1/d) Tr(Q^{ab} Q^{cd}) = lambda_j P2^{abcd},

where P2 is the standard spin-2 projector, and lambda_j is the quadrupole norm
coefficient:

  lambda_j = 2 j(j+1)(2j-1)(2j+3) / 45.

In particular, lambda_{1/2} = 0, confirming Q^{ab} = 0 at j = 1/2, and
lambda_j > 0 for all j >= 1.


C.4-C.5 Extracting A and B (Direct Method)

By SO(3) isotropy, T_TO is completely determined by two independent components.

Step 1: Compute B.
Set i=k=x, j=l=y in T_TO^{ijkl} = (1/d) Tr(J^i J^k J^j J^l):

  B = (1/d) Tr(J^x J^x J^y J^y).

Using the Casimir relation and SO(3) isotropy:

  B(j) = j(j+1)(2j^2 + 2j + 1) / 30.

Step 2: Compute A via the fourth-moment identity.
By SO(3) isotropy, A + 2B = (1/d) Tr((J^x)^4) = j(j+1)(3j^2+3j-1)/15.
Therefore:

  A(j) = j(j-1)(j+1)(j+2) / 15,
  B(j) = C(j) = j(j+1)(2j^2 + 2j + 1) / 30,
  A(j) + B(j) = j(j+1)(2j-1)(2j+3) / 30 =: F(j)/30.

Normalisation check. At j=1: F(1)/30 = 10/30 = 1/3. Direct: A+B = 0+1/3 = 1/3. ok.


C.6 Summary and the E2 Condition

Theorem (closed-form coefficients). In the irreducible spin-j representation:

  A(j) = j(j-1)(j+1)(j+2) / 15
  B(j) = C(j) = j(j+1)(2j^2+2j+1) / 30
  A(j) + B(j) = F(j)/30,   F(j) = j(j+1)(2j-1)(2j+3).

The E2 condition B = -A is equivalent to A + B = 0, i.e., F(j) = 0. Factoring:

  F(j) = 0  iff  j = 0 (trivial), j = -1 (unphysical), j = 1/2, or j = -3/2.

The root j = -3/2 is excluded: no finite-dimensional unitary representation of
SU(2) exists for that value (the inner product fails to be positive-definite).
Only j = 1/2 is a valid physical solution.


C.7 Connection to the Quadrupole: E2-Nullification

The coefficient F(j)/30 admits a direct interpretation in terms of Q^{ab}:

  A + B = F(j)/30  and  lambda_j = 2F(j) / (3 j(j+1))   for j > 0,

so lambda_j = 0 iff F(j) = 0. The full chain of equivalences is:

  E2  <-->  A+B = 0  <-->  F(j) = 0  <-->  lambda_j = 0
       <-->  Q^{ab} = 0  <-->  m_2(rho_j) = 0.

The equivalence Q^{ab} = 0 <--> lambda_j = 0 follows from:
- If Q^{ab} = 0 then (1/d)Tr(Q^{ab}Q^{ab}) = 0 so lambda_j = 0.
- If lambda_j = 0 then (1/d)Tr(Q^{ab}Q^{ab}) = 0 and since <.,.> is
  positive-definite (j physical), Q^{ab} = 0.
- Since J^a generate an irreducible representation, rho_2 absent in
  Sym^2(rho_j) iff Q^{ab} := Pi_2(J^a J^b) = 0.

This chain shows that the E2 condition is not merely a numerical coincidence
but reflects the structural absence of the spin-2 sector in the operator
algebra. This is the content of the E2-Nullification theorem presented in
Section 5A.
