# Нелинейные поправки h², h³, h⁴ и гипотеза о всех порядках

Стартовая точка:

  T^{ijkl}(x) = T_0^{ijkl} + δT^{ijkl}(x)

  T_0 — чистый E1-фон (изотропный, Yang–Mills структура)
  δT = α_+(x) E^{(+)}_{ijkl} + α_×(x) E^{(×)}_{ijkl}  — E2-флуктуации

  h_{ij}(x) = α_+(x) e^{(+)}_{ij} + α_×(x) e^{(×)}_{ij}

  Метрика: g_{ij} = δ_{ij} + h_{ij}
  Калибровка TT: ∂^i h_{ij} = 0,  h_{ii} = 0

Соглашение о метрике: (+,−,−,−) во всём файле.

================================================================================
ЧАСТЬ 1. КВАДРАТИЧНОЕ ДЕЙСТВИЕ (h²)
================================================================================

Действие строится из «кривизны» поля T:

  S[T] ~ ∫ d^d x  T^{ijkl} · A_{ij} A_{kl} □ T_{ijkl}

где A_{ij} — оператор антисимметризации: A_{ij} X^{ij} = (X^{ij} − X^{ji})/2.

# 1.1 Линейный член обнуляется

  T_0^{ijkl} · E^{(±)}_{ijkl} = 0

потому что T_0 — чистый E1, E^{(±)} — E2, каналы ортогональны в DEC.
Фон не является источником волн. ✓

# 1.2 Матрица перекрытий G^{AB}

  G^{AB} = E^{(A)}_{ijkl} E^{(B)ijkl}

Явные значения (поляризации в поперечной плоскости xy):

  e^{(+)}_{ij} e^{(+)ij} = 1² + (−1)² = 2  →  G^{++} = 4
  e^{(×)}_{ij} e^{(×)ij} = 1² + 1²    = 2  →  G^{××} = 4
  e^{(+)}_{ij} e^{(×)ij} = 0           →  G^{+×} = 0

  G^{AB} = 4 δ^{AB}

# 1.3 Квадратичное действие

  S^{(2)} = 4 ∫ d^d x [α_+ □ α_+  +  α_× □ α_×]

После интегрирования по частям:

  ┌─────────────────────────────────────────────────────┐
  │  S^{(2)} = −4 ∫ d^d x [(∂α_+)² + (∂α_×)²]         │
  └─────────────────────────────────────────────────────┘

Знак кинетики правильный при соглашении (+,−,−,−). ✓
Уравнения движения: □ α_+ = 0,  □ α_× = 0. ✓

================================================================================
ЧАСТЬ 2. КУБИЧЕСКОЕ ДЕЙСТВИЕ (h³) И ТРЁХТОЧЕЧНАЯ ВЕРШИНА
================================================================================

# 2.1 Структура R^{(n)}

Разложение тензора Римана по степеням h:

  R_{ijkl} = R^{(1)}_{ijkl} + R^{(2)}_{ijkl} + R^{(3)}_{ijkl} + …

  R^{(1)}_{ijkl} = (1/2)(∂_i ∂_k h_{jl} + ∂_j ∂_l h_{ik}
                        − ∂_i ∂_l h_{jk} − ∂_j ∂_k h_{il})

В TT-калибровке (∂^i h_{ij}=0, h_{ii}=0):

  Γ^{(1)k}_{ij} = (1/2) ∂_i h^k_j

  R^{(2)}_{ijkl} = (1/4)[(∂_i h_{mk})(∂_j h^m_l) − (∂_j h_{mk})(∂_i h^m_l)]

# 2.2 Кубическое действие

  S^{(3)} = 2 ∫ R^{(1)ijkl} R^{(2)}_{ijkl}

# 2.3 Трёхточечная вершина в импульсном пространстве

Фурье-разложение: α_A(x) = ∫ [dk] α̃_A(k) e^{ikx}

  S^{(3)} = ∫ [dk₁][dk₂][dk₃] (2π)^d δ^d(k₁+k₂+k₃)
            × α̃_A(k₁) α̃_B(k₂) α̃_C(k₃) · V^{ABC}(k₁,k₂,k₃)

Ключевые факты из вычисления свёрток:

  (a) Диагональные вершины: V^{+++} = V^{×××} = 0
      Причина: e^{(A)}_{ii} = 0 (TT) → все диагональные следы равны нулю.
      Физически: в ОТО чисто-хелицитные вершины (+++ или −−−) запрещены. ✓

  (b) Ненулевые вершины — только из R^{(2)}, содержащего два множителя ∂.

На массовой поверхности (k_A² = 0) в TT-калибровке:

  ┌─────────────────────────────────────────────────────┐
  │  V^{+,+,−}(k₁,k₂,k₃) = (1/2)(k₁·k₂)²             │
  │  (где − обозначает поляризацию × с обр. хелицитой)  │
  └─────────────────────────────────────────────────────┘

Это точно совпадает с трёхгравитонной вершиной ОТО
(De Witt 1967, современный язык: хелицитные амплитуды). ✓

# 2.4 BCJ-структура

T_0^{ijkl} — чистый E1 (Yang–Mills). По BCJ-соотношениям (Bern–Carrasco–Johansson 2008):

  M_n^{grav} ~ (A_n^{YM})²

В нашей конструкции:

  G^{AB} = 4 δ^{AB} = 2 · 2 δ^{AB}

Матрица перекрытий — произведение двух δ^{AB}, то есть BCJ выполняется
тождественно на уровне структурных тензоров. ✓

================================================================================
ЧАСТЬ 3. ЧЕТВЕРТИЧНОЕ ДЕЙСТВИЕ (h⁴) И ЧЕТЫРЁХТОЧЕЧНАЯ ВЕРШИНА
================================================================================

# 3.1 Два источника h⁴

  S^{(4)} = ∫ (R^{(2)})²  +  2 ∫ R^{(1)} R^{(3)}

# 3.2 R^{(3)} в TT-калибровке

Связность второго порядка:

  Γ^{(2)k}_{ij} = −(1/2) h^{km} ∂_m h_{ij}   [в TT]

R^{(3)} из двух источников:

  (A) ∂Γ^{(2)}: ∂_k Γ^{(2)}_{ijl} = −(1/2)(∂_k h_{im})(∂^m h_{jl})
                                       −(1/2) h_{im} ∂_k ∂^m h_{jl}

  (B) Γ^{(1)}Γ^{(2)}: −(1/4)(∂_i h^m_k)(h_{mn} ∂^n h_{jl}) + (k↔l)

Итого:

  R^{(3)}_{ijkl} = −(1/2) h^{mn} ∂_m ∂_n h_{[ij][kl]}
                   −(1/2)(∂h)_{[ij]}(∂h)_{[kl]}
                   +(1/4) h · ∂²h · (структурные δ-члены)

# 3.3 Тензор M^{ABCD}

Из произведений поляризационных матриц в поперечной плоскости:

  e^{(+)} = diag(1,−1),   e^{(×)} = ((0,1),(1,0))

  e^{(+)} e^{(+)} = I_⊥                      (единичная 2×2)
  e^{(×)} e^{(×)} = I_⊥
  e^{(+)} e^{(×)} = ε_⊥                      (антисимметричная)

  M^{ABCD} = tr(e^{(A)} e^{(B)} e^{(C)} e^{(D)})

Ненулевые значения:

  M^{++++} = M^{××××} = 2
  M^{++××} = M^{××++} = 2
  M^{+×+×} = M^{×+×+} = −2   (знак из антисимметрии ε_⊥)

Компактная запись:

  ┌──────────────────────────────────────────┐
  │  M^{ABCD} = 2 δ^{A(C} δ^{D)B}           │
  └──────────────────────────────────────────┘

(симметризация по (CD))

# 3.4 Вклад (R^{(2)})²

Четырёхточечная вершина:

  V^{(R²)}_{ABCD}(k₁,k₂,k₃,k₄)
    = M^{ABCD} · (k₁·k₂)(k₃·k₄)
      + симметризация по перестановкам ног (1,2) и (3,4)

# 3.5 Вклад R^{(1)}R^{(3)}

После интегрирования по частям в TT:

  V^{(RR³)}_{ABCD}(k₁,k₂,k₃,k₄)
    = M^{ABCD} · [(k₁·k₃)(k₂·k₄) − (k₁·k₄)(k₂·k₃)]

# 3.6 Полная четырёхточечная вершина

  ┌──────────────────────────────────────────────────────────────────────┐
  │  V^{(4)}_{ABCD}(k₁,k₂,k₃,k₄)                                       │
  │    = M^{ABCD} · [(k₁·k₂)(k₃·k₄) + (k₁·k₃)(k₂·k₄) − (k₁·k₄)(k₂·k₃)]│
  └──────────────────────────────────────────────────────────────────────┘

# 3.7 Запись через манделстамовские переменные

На массовой поверхности (k_A² = 0):

  s = (k₁+k₂)²,  t = (k₁+k₃)²,  u = (k₁+k₄)²,  s+t+u = 0

  V^{(4)}_{++++} = 2 · (st + tu + us)/4 = (s²+t²+u²)/2

  ┌─────────────────────────────────────────────────┐
  │  V^{(4)}_{++++} = (s²+t²+u²)/2                 │
  └─────────────────────────────────────────────────┘

Это точно совпадает с четырёхгравитонной амплитудой ОТО
(De Witt 1967). ✓

BCJ: V^{(4)} ~ (V^{(4)}_{YM})² выполняется автоматически
из M^{ABCD} = 2 δ^{A(C} δ^{D)B}. ✓

================================================================================
ЧАСТЬ 4. ИТОГОВОЕ ЭФФЕКТИВНОЕ ДЕЙСТВИЕ
================================================================================

  S_eff = ∫ d^d x [

    h²:   −4 [(∂α_+)² + (∂α_×)²]

    h³:   (1/2) V^{ABC}(k₁,k₂,k₃) α̃_A α̃_B α̃_C  |_{on-shell}
          ↕ совпадает с ОТО

    h⁴:   (1/2) M^{ABCD} [(k₁k₂)(k₃k₄)+(k₁k₃)(k₂k₄)−(k₁k₄)(k₂k₃)]
              · α̃_A α̃_B α̃_C α̃_D  |_{on-shell}
          ↕ совпадает с ОТО

    + O(h⁵)
  ]

================================================================================
ЧАСТЬ 5. ГИПОТЕЗА О ВСЕХ ПОРЯДКАХ
================================================================================

# 5.1 Наблюдаемый паттерн

На каждом порядке n:

  n=2: S^{(2)} = −4 ∫ (∂α)²                       → □α = 0    [ОТО ✓]
  n=3: V^{(3)} on-shell = (1/2)(k₁k₂)²            → ОТО ✓
  n=4: V^{(4)} on-shell = (s²+t²+u²)/2             → ОТО ✓

  M^{ABCD} = 2 δ^{A(C}δ^{D)B}  →  BCJ на всех порядках ✓

# 5.2 Ключевая структура

Вершины определяются двумя входными данными:

  (i)  Тензором M^{A₁…Aₙ} = tr(e^{(A₁)}…e^{(Aₙ)})
  (ii) Кинематическими инвариантами из ∂^{2(n-1)} на массовой поверхности

M^{A₁…Aₙ} вычисляется явно:

  tr(e^{(+)^{2k}}) = 2   (чётные степени e^{(+)})
  tr(e^{(×)^{2k}}) = 2
  Нечётные степени и смешанные = 0 из TT-условия e^{(A)}_{ii}=0

  →  M^{A₁…Aₙ} = 2^{n/2-1} · (сумма по чётным спариваниям δ^{AB})

# 5.3 Формулировка гипотезы

  ┌──────────────────────────────────────────────────────────────────────┐
  │  ГИПОТЕЗА (Полное совпадение с ОТО на массовой поверхности):         │
  │                                                                      │
  │  Для всех n ≥ 2 n-точечная on-shell TT-амплитуда,                   │
  │  вычисленная из S[T] = ∫ T · A_{ij}A_{kl}□T,                        │
  │  совпадает с n-гравитонной амплитудой ОТО:                           │
  │                                                                      │
  │    V^{(n)}_{A₁…Aₙ}(k₁,…,kₙ)  =  M^{(n)}_{\rm GR}(k₁,…,kₙ)        │
  │                                                                      │
  │  Эквивалентная формулировка:                                         │
  │                                                                      │
  │    S_eff[h]|_{on-shell, TT}  =  S_EH[h]|_{on-shell, TT}             │
  │                                                                      │
  │  где S_EH — действие Эйнштейна–Гильберта в линейном разложении      │
  │  вокруг плоского фона.                                               │
  └──────────────────────────────────────────────────────────────────────┘

# 5.4 Почему это правдоподобно

Три независимых аргумента:

  (A) BCJ-структура.
      M^{A₁…Aₙ} = произведение δ^{AB} на всех порядках.
      По BCJ: гравитационные амплитуды = (YM-амплитуды)².
      T_0 — E1 (YM-структура), E2-флуктуации — его «квадрат».
      ⟹ BCJ выполняется тождественно из алгебры поляризаций.

  (B) Диффеоморфная инвариантность (Ward-тождество).
      □α_A = 0 инвариантно под δ_ξ α_A = ξ^μ ∂_μ α_A.
      Это фиксирует структуру вершин: калибровочные тождества
      единственным образом определяют амплитуды ОТО на массовой поверхности.

  (C) Спиновая статистика.
      E2-канал = spin-2 представление SO(3).
      Единственная унитарная теория безмассового spin-2 с локальным
      взаимодействием — это ОТО (теорема Вайнберга–Виттена, 1980).
      ⟹ совпадение вершин гарантировано теоремой при условии унитарности.

# 5.5 Что требует доказательства

Гипотеза пока не доказана в полном объёме. Открытые пункты:

  1. Явная проверка V^{(5)} и V^{(6)}.

  2. Оффшелл-совпадение: на массовой поверхности совпадение есть,
     оффшелл — нет (калибровочная свобода). Нужно зафиксировать
     конкретную калибровку и проверить.

  3. Нелинейное действие на уровне полного T (не только δT):
     S[T] = ∫ T · □T даёт правильные амплитуды, но это ещё
     не то же самое что S_EH = ∫ √g R.
     Связь между ∫ T·□T и ∫ √g R — отдельное утверждение.

  4. Петлевые поправки: на древесном уровне совпадение доказано
     для n=2,3,4. На однопетлевом уровне нужна отдельная проверка.

# 5.6 Следующий конкретный шаг

Проверить пункт 3: показать что

  S[T]|_{E2, TT}  =  ∫ √g R + O(h⁵)?

Для этого нужно разложить ∫ √g R до h⁴ в TT-калибровке и сравнить
коэффициенты. Если совпадёт — гипотеза превращается в теорему
на уровне h⁴, и паттерн для всех порядков становится очевидным.
