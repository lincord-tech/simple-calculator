ВЫЧИСЛЕНИЯ: T_cross, квадрупольный канал, выход на E2

Базовые формулы (spin-j, SU(2)):

  A(j) = j(j-1)(j+1)(j+2) / 15
  B(j) = j(j+1)(2j^2 + 2j + 1) / 30
  F(j) = j(j+1)(2j-1)(2j+3)

DEC-разложение изотропного тензора ранга 4:
  T^ijkl = A delta^ij delta^kl + B delta^ik delta^jl + C delta^il delta^jk

E2-условие (Einstein-Hilbert): A = 0, B = -C.
E1-условие (Yang-Mills):       A = -C, B = 0.

Проектор на симметричные бесследовые тензоры ранга 2 (spin-2):
  P2^abcd = (1/2)(delta^ac delta^bd + delta^ad delta^bc) - (1/3) delta^ab delta^cd

Квадрупольная нормировка:
  lambda_j = 2 j(j+1)(2j-1)(2j+3) / 45

--------------------------------------------------------------------------------
ЧАСТЬ 1. ТАБЛИЦА A, B, A+B ДЛЯ T_TO
--------------------------------------------------------------------------------

  T_TO^ijkl = (1/d) Tr(J^i J^k J^j J^l)

  j=1/2 : A = -1/16,  B = C =  1/16,  A+B = 0        (F/30 = 0)
  j=1   : A =  0,     B = C =  1/3,   A+B = 1/3      (F/30 = 1/3)
  j=3/2 : A =  7/16,  B = C = 17/16,  A+B = 3/2      (F/30 = 3/2)
  j=2   : A =  8/5,   B = C = 13/5,   A+B = 21/5     (F/30 = 21/5)

E2-условие B = -A выполняется только при j = 1/2 (A+B = 0 <=> F(j) = 0).

--------------------------------------------------------------------------------
ЧАСТЬ 2. КОНСТРУКЦИЯ T_cross И ТЕНЗОР ПОПРАВКИ Delta
--------------------------------------------------------------------------------

  T_cross^ijkl = T_TO^ijkl + G2 * lambda_j * Delta^ijkl

где

  Delta^ijkl = P2^ikab P2^jlcd T_TO^acbd

# Вывод Delta

A-часть T_TO (с delta^ac delta^bd) уничтожается одним из проекторов P2
(P2^ikab delta^ik = 0 — проектор бесследовый). Остаётся только B-часть:

  T_TO^acbd -> B(j) * delta^ad delta^bc

Подставляем:
  Delta^ijkl = B(j) * P2^ikab P2^jlcb

По SO(3)-инвариантности результат пропорционален единственной возможной структуре:
  P2^ikab P2^jlcb = mu * (delta^ij delta^kl - (2/3) delta^ik delta^jl + delta^il delta^jk)

Коэффициент mu находим через след. Берём свёртку с delta^ij delta^kl.

Правая часть:
  delta^ij delta^kl * (delta^ij delta^kl - (2/3) delta^ik delta^jl + delta^il delta^jk)
  = delta^ij delta^ij * delta^kl delta^kl - (2/3) delta^ij delta^kl delta^ik delta^jl
    + delta^ij delta^kl delta^il delta^jk
  = 9 - (2/3)*3 + 3 = 10

(здесь использована размерность d=3: delta^ii = 3)

Левая часть:
  delta^ij delta^kl P2^ikab P2^jlcb = dim(spin-2) = 5

(след произведения проекторов равен размерности подпространства spin-2 в Mat(3x3))

Из 10 * mu = 5 получаем mu = 1/2.

Поскольку Delta^ijkl = B(j) * mu * (...), определяем:

  kappa(j) = mu * B(j) = B(j) / 2 = j(j+1)(2j^2 + 2j + 1) / 60

Итого:

  Delta^ijkl = kappa(j) * (delta^ij delta^kl - (2/3) delta^ik delta^jl + delta^il delta^jk)

DEC-коэффициенты Delta:
  A_Delta = kappa(j)
  B_Delta = -(2/3) kappa(j)
  C_Delta = kappa(j)

# Числовые значения kappa(j)

  j=1   : kappa = 1*2*5 / 60 = 1/6
  j=3/2 : j(j+1) = 15/4,  2j^2+2j+1 = 17/2
          kappa = (15/4)*(17/2) / 60 = 255/480 = 17/32

# Проверка DEC-коэффициентов Delta

  j=1   : A_Delta = 1/6,    B_Delta = -1/9,    C_Delta = 1/6     [верно]
  j=3/2 : A_Delta = 17/32,  B_Delta = -17/48,  C_Delta = 17/32   [верно]

ПРИМЕЧАНИЕ: в рабочих записях j=3/2 используется Delta с коэффициентами (3/4, -1/2, 3/4).
Это другая нормировка: (3/4) / (17/32) = 24/17. Структура направления универсальная,
масштаб отличается. Все последующие вычисления с (3/4, -1/2, 3/4) для j=3/2 внутренне
согласованы — параметр g просто поглощает нормировочный множитель.

# Универсальное направление

Для любого j структура вектора Delta в DEC-пространстве одинакова:
  (A_Delta, B_Delta, C_Delta) = kappa(j) * (1, -2/3, 1)

Зависимость от j только через общий масштаб kappa(j).

  P2 P2 T_TO = kappa(j) * V_universal,
  V_universal = (1, -2/3, 1)  в DEC

--------------------------------------------------------------------------------
ЧАСТЬ 3. E2-КОМПОНЕНТА ДЛЯ j=3/2
--------------------------------------------------------------------------------

Параметр g = G2 * lambda_j.

Полные коэффициенты при Delta = (3/4, -1/2, 3/4):

  A_tot(g) = 7/16 + g * (3/4)
  B_tot(g) = 17/16 + g * (-1/2)
  C_tot(g) = 17/16 + g * (3/4)

E2-проекция (координата вдоль E2 в разложении по базису {E_trace, E1, E2}):

  z(g) = (2 B_tot - A_tot - C_tot) / 3

Подставляем:
  числовая часть: (2*(17/16) - 7/16 - 17/16) / 3 = (34/16 - 7/16 - 17/16) / 3
                  = (10/16) / 3 = 10/48 = 5/24
  часть при g:    (2*(-1/2) - 3/4 - 3/4) / 3 = (-1 - 3/2) / 3 = (-5/2) / 3 = -5/6

  z(g) = 5/24 - (5/6) * g

lambda_j(3/2) = 2*(3/2)*(5/2)*(2)*(5) / 45 = 2*(15/4)*2*(5/2)/(wait)

Явный расчёт:
  lambda_3/2 = 2 * (3/2)(5/2)(2)(5) / 45 = 2*(75/4) / 45 = 150/180 = 5/6... 

Пересчёт:
  j=3/2: j(j+1) = 15/4,  (2j-1) = 2,  (2j+3) = 6
  F(3/2) = (15/4)*2*6 = 45
  lambda_3/2 = 2*F(3/2)/45 * (1/(2j-1)(2j+3))...

Явная формула:
  lambda_j = 2 j(j+1)(2j-1)(2j+3) / 45
  lambda_3/2 = 2*(15/4)*2*6 / 45 = 2*45 / 45 = 2

Физический диапазон: G2 in [-1/3, 1], поэтому g = G2*lambda_j in [-2/3, 2].

  z(g) = 5/24 - (5/6)*g  -- убывает по g, максимум при g = -2/3

  z_max = 5/24 - (5/6)*(-2/3) = 5/24 + 10/18 = 5/24 + 20/36 = 15/72 + 40/72 = 55/72
  z_min = 5/24 - (5/6)*(2) = 5/24 - 5/3 = 5/24 - 40/24 = -35/24

  При g=0 (T_TO): z_TO = 5/24

Сравнение:
  z_max / z_TO = (55/72) / (5/24) = (55/72) * (24/5) = 55*24 / (72*5) = 1320/360 = 11/3 ≈ 3.67

--------------------------------------------------------------------------------
ЧАСТЬ 4. ОДНОГО ПАРАМЕТРА g НЕДОСТАТОЧНО ДЛЯ ЧИСТОГО E2
--------------------------------------------------------------------------------

Условия чистого E2-луча:
  (1)  A_tot = 0
  (2)  B_tot = -C_tot

Из (1): g = -A(j) / a   (a = A_Delta = компонента Delta по delta^ij delta^kl)
Из (2): g = -2B(j) / (a + b)   (b = B_Delta)

Совместность: A(j)/a = 2B(j)/(a+b)

Подставляем структуру (a, b) = kappa(j) * (1, -2/3):
  a + b = kappa(j) * (1 - 2/3) = kappa(j) / 3

  Условие: A(j) * (kappa(j)/3) = 2B(j) * kappa(j)
  => A(j)/3 = 2B(j)
  => A(j) = 6 B(j)

Проверяем:
  A/B = [j(j-1)(j+1)(j+2)/15] / [j(j+1)(2j^2+2j+1)/30]
       = 2(j-1)(j+2) / (2j^2+2j+1)

Уравнение A = 6B:
  2(j-1)(j+2) = 6(2j^2+2j+1)
  2(j^2 + j - 2) = 6(2j^2 + 2j + 1)
  2j^2 + 2j - 4 = 12j^2 + 12j + 6
  0 = 10j^2 + 10j + 10
  j^2 + j + 1 = 0
  Дискриминант: 1 - 4 = -3 < 0

Вещественных решений нет. Вывод: одного параметра g недостаточно для точного
попадания в E2-луч ни при каком физическом j >= 1.

Проверка для конкретных j:
  j=1   : A/B = 0   (не равно 6)
  j=3/2 : A/B = 7/17 ≈ 0.41  (не равно 6)

Геометрически: T_TO и T_TO + g*Delta движутся вдоль прямой,
которая не пересекает луч E2 в пространстве DEC.

--------------------------------------------------------------------------------
ЧАСТЬ 5. ЗАМКНУТОЕ РЕШЕНИЕ С ДВУМЯ ПАРАМЕТРАМИ (g, epsilon)
--------------------------------------------------------------------------------

Добавляем анизотропный канал:
  T_total = T_TO + g * Delta + epsilon * V_aniso
  V_aniso = (p, q, r) в DEC

Полные коэффициенты:
  A_tot = A + g*kappa + epsilon*p
  B_tot = B + g*(-(2/3)*kappa) + epsilon*q
  C_tot = B + g*kappa + epsilon*r

Условия E2: A_tot = 0, B_tot = -C_tot.

Уравнение (1): A + g*kappa + epsilon*p = 0
  => epsilon = -(A + g*kappa) / p    [при p != 0]

Уравнение (2): B + g*(-(2/3)*kappa) + epsilon*q = -(B + g*kappa + epsilon*r)
  => 2B + g*kappa*(1/3) + epsilon*(q + r) = 0

Подставляем epsilon из (1) в (2):
  2B + g*kappa*(1/3) - (A + g*kappa)*(q+r)/p = 0

Умножаем на p:
  2Bp + g*kappa*p/3 - (A + g*kappa)*(q+r) = 0
  2Bp + g*kappa*p/3 - A*(q+r) - g*kappa*(q+r) = 0
  g*kappa * (p/3 - (q+r)) + (2Bp - A*(q+r)) = 0

  g(j) = [A(j)*(q+r) - 2*B(j)*p] / [kappa(j) * (p/3 - (q+r))]
  epsilon(j) = -(A(j) + g(j)*kappa(j)) / p

# Выбор V_aniso = (1, 0, -1): p=1, q=0, r=-1, S = q+r = -1

  g(j) = [A*(-1) - 2B*1] / [kappa*(1/3 - (-1))]
       = -(A + 2B) / [kappa * (4/3)]
       = -3(A + 2B) / (4*kappa)

  epsilon(j) = -(A + g*kappa)
             = -(A + (-3(A+2B)/4))
             = -(A - 3A/4 - 3B/2)
             = -(A/4 - 3B/2)
             = 3B/2 - A/4

Числовые значения:

  j=1:
    A = 0,  B = 1/3,  kappa = 1/6
    g = -3*(0 + 2/3) / (4*(1/6)) = -3*(2/3)/(2/3) = -3
    epsilon = 3/2*(1/3) - 1/4*0 = 1/2

  j=3/2:
    A = 7/16,  B = 17/16,  kappa = 17/32
    g = -3*(7/16 + 17/8) / (4*(17/32))
      = -3*(7/16 + 34/16) / (17/8)
      = -3*(41/16) / (17/8)
      = -3*(41/16)*(8/17)
      = -3*41 / (2*17)
      = -123/34
    epsilon = 3/2*(17/16) - 1/4*(7/16) = 51/32 - 7/64 = 102/64 - 7/64 = 95/64

--------------------------------------------------------------------------------
ЧАСТЬ 6. АНАЛИТИЧЕСКАЯ ФОРМУЛА kappa(j)
--------------------------------------------------------------------------------

  kappa(j) = B(j) / 2 = j(j+1)(2j^2 + 2j + 1) / 60

Проверка:
  j=1:   kappa = 1*2*5/60 = 10/60 = 1/6     [верно]
  j=3/2: kappa = (15/4)*(17/2)/60 = 255/480 = 17/32   [верно]
  j=1/2: kappa = (3/4)*(9/2)/60 = 27/8/60 = 27/480 = 9/160 ...

Пересчёт j=1/2:
  j(j+1) = (1/2)(3/2) = 3/4
  2j^2+2j+1 = 2*(1/4) + 2*(1/2) + 1 = 1/2 + 1 + 1 = 5/2
  kappa = (3/4)*(5/2)/60 = (15/8)/60 = 15/480 = 1/32

  Проверка через B/2: B(1/2) = (3/4)*(5/2)/30 = (15/8)/30 = 15/240 = 1/16
  kappa = 1/16 / 2 = 1/32   [верно]

Асимптотика: kappa(j) ~ j^4/30 при j -> infinity.

--------------------------------------------------------------------------------
ЧАСТЬ 7. ТЕНЗОР ЭФФЕКТИВНОГО ЭНЕРГИИ-ИМПУЛЬСА
--------------------------------------------------------------------------------

Определение через контракцию:
  T_eff^ij = T_cross^ikjk  (суммирование по k)

Подстановка DEC-разложения T_cross = (A_tot, B_tot, C_tot):

  (i)  A_tot * delta^ij * delta^kk = 3 A_tot * delta^ij
  (ii) B_tot * delta^ik * delta^jk = B_tot * delta^ij
  (iii) C_tot * delta^ik * delta^jk = C_tot * delta^ij

  T_eff^ij = (3 A_tot + B_tot + C_tot) * delta^ij

Коэффициенты T_cross:
  A_tot = A(j) + g*kappa(j) + epsilon*p
  B_tot = B(j) - (2/3)*g*kappa(j) + epsilon*q
  C_tot = B(j) + g*kappa(j) + epsilon*r

Сумма:
  3 A_tot + B_tot + C_tot
  = 3A + 2B + g*kappa*(3 - 2/3 + 1) + epsilon*(3p + q + r)
  = 3A + 2B + g*kappa*(10/3) + epsilon*(3p + q + r)

Базовая часть:
  3A + 2B = (3/15)*j(j-1)(j+1)(j+2) + (2/30)*j(j+1)(2j^2+2j+1)
           = (j(j+1)/15) * [3(j-1)(j+2) + (2j^2+2j+1)]

  3(j-1)(j+2) = 3j^2 + 3j - 6
  3j^2 + 3j - 6 + 2j^2 + 2j + 1 = 5j^2 + 5j - 5 = 5(j^2 + j - 1)

  3A + 2B = j(j+1)(j^2 + j - 1) / 3

Упрощение коэффициента при g:
  (10/3) * (1/60) = 10/180 = 1/18

Итоговая формула:

  T_eff^ij = rho_eff(j) * delta^ij

  rho_eff(j) = j(j+1)(j^2+j-1)/3
               + g * j(j+1)(2j^2+2j+1) / 18
               + epsilon * (3p + q + r)

Числовые значения rho_eff при g=0, epsilon=0:
  j=1/2 : rho = -1/16      (отрицательное, квазичастица)
  j=1   : rho =  2/3
  j=3/2 : rho = 55/16

Асимптотика: rho_eff ~ j^4/3 при j -> infinity.

При V_aniso = (1, 0, -1): 3p + q + r = 3 + 0 - 1 = 2, поэтому вклад epsilon*2.



ВЫЧИСЛЕНИЯ: континуальный предел j→∞, расстояние до E2, линеаризация

ОБОЗНАЧЕНИЯ

  A(j) = j(j-1)(j+1)(j+2) / 15
  B(j) = j(j+1)(2j^2+2j+1) / 30
  kappa(j) = j(j+1)(2j^2+2j+1) / 60  [= B(j)/2]
  epsilon(j) = 3B(j)/2 - A(j)/4 = j(j+1)(j^2+j+1) / 12  [V_aniso=(1,0,-1)]
  g(j) = -3(A(j)+2B(j)) / (4*kappa(j)) = -j(j+1)(3j^2+3j-1) / (20*kappa(j))

DEC-пространство: T^ijkl = A delta^ij delta^kl + B delta^ik delta^jl + C delta^il delta^jk

T_cross(g, epsilon) = T_TO + g * Delta + epsilon * V_aniso
  Delta DEC: (kappa, -2/3*kappa, kappa)
  V_aniso DEC: (1, 0, -1)  [невырожденный выбор]

--------------------------------------------------------------------------------
ЧАСТЬ 1. АСИМПТОТИКА ПРИ j → ∞
--------------------------------------------------------------------------------

# 1.1 Ведущие коэффициенты

  A(j)       ~ j^4/15  + O(j^3)
  B(j)       ~ j^4/15  + O(j^3)   [то же ведущее, разные следующие]
  A + 2B     ~ 3j^4/15 = j^4/5 + O(j^3)
  kappa(j)   ~ j^4/30  + O(j^3)
  epsilon(j) ~ j^4/12  + O(j^3)

# 1.2 Предел g(j)

  g(j) = -3(A+2B) / (4*kappa)
       ~ -3 * (j^4/5) / (4 * j^4/30)
       = -3 * (1/5) * (30/4)
       = -3 * 6/4
       = -9/2

  lim_{j→∞} g(j) = -9/2

  [ИСПРАВЛЕНИЕ: в исходном файле ошибочно указано g→-3/10.
   Ошибка: файл использовал отношение (1/5)/(1/20)=4, а не (1/5)/(1/30)=6.
   Правильное ведущее kappa ~ j^4/30, не j^4/20.]

# 1.3 Масштабирование всех вкладов

  T_TO         ~ j^4/15 * (1,1,1) в DEC
  g * Delta    ~ (-9/2) * (j^4/30) * (1,-2/3,1) ~ j^4 * (-3/20, 1/10, -3/20)
  eps * V_aniso~ (j^4/12) * (1,0,-1) ~ j^4 * (1/12, 0, -1/12)

  Все вклады ~ j^4. После нормировки T_cross / j^4 имеет конечный предел.

# 1.4 Нормированный предел

  T_tilde = T_cross / j^4

  A_tilde = 1/15 + (-9/2)*(1/30) + (1/12)*1 = 1/15 - 3/20 + 1/12
           = 4/60 - 9/60 + 5/60 = 0
  B_tilde = 1/15 + (-9/2)*(-2/3)*(1/30) = 1/15 + 1/10 = 2/30 + 3/30 = 1/6
  C_tilde = 1/15 + (-9/2)*(1/30) + (1/12)*(-1) = 1/15 - 3/20 - 1/12
           = 4/60 - 9/60 - 5/60 = -10/60 = -1/6

  T_tilde = (0, 1/6, -1/6) ∝ (0, 1, -1) = E2

  Нормированный предел точно лежит на E2-луче. ✓

--------------------------------------------------------------------------------
ЧАСТЬ 2. ОРТОГОНАЛЬНЫЙ БАЗИС В DEC ≅ R^3
--------------------------------------------------------------------------------

Базисные векторы:
  e2 = (0, 1, -1)   — направление E2
  e1 = (1, 1, 1)    — trace-направление
  e0 = (2, -1, -1)  — ортогональное дополнение

Проверка ортогональности:
  e2 · e1 = 0 + 1 - 1 = 0   ✓
  e2 · e0 = 0 - 1 + 1 = 0   ✓
  e1 · e0 = 2 - 1 - 1 = 0   ✓

Нормы:
  |e1|^2 = 1 + 1 + 1 = 3
  |e0|^2 = 4 + 1 + 1 = 6
  |e2|^2 = 0 + 1 + 1 = 2

Разложение T = (A, B, C):
  T = alpha * e2 + beta * e1 + gamma * e0

Проекции (через скалярные произведения):
  alpha = (B - C) / 2
  beta  = (A + B + C) / 3
  gamma = (2A - B - C) / 6

Проверка обратного восстановления:
  A = beta + 2*gamma     = (A+B+C)/3 + (2A-B-C)/3 = A  ✓
  B = alpha + beta - gamma = (B-C)/2 + (A+B+C)/3 - (2A-B-C)/6
    = (3(B-C) + 2(A+B+C) - (2A-B-C)) / 6
    = (3B-3C + 2A+2B+2C - 2A+B+C) / 6
    = 6B/6 = B  ✓
  C = -alpha + beta - gamma: аналогично = C  ✓

Условие точного попадания в E2:
  beta = 0  и  gamma = 0

--------------------------------------------------------------------------------
ЧАСТЬ 3. РАССТОЯНИЕ ДО E2-ЛУЧА
--------------------------------------------------------------------------------

Расстояние от точки T = (A, B, C) до прямой через начало координат с направлением e:

  d^2 = |T|^2 - (T·e)^2 / |e|^2

E2-луч: e = (0, 1, -1), |e|^2 = 2.

  T · e = B - C

  d^2 = A^2 + B^2 + C^2 - (B-C)^2 / 2

# 3.1 Случай только квадруполя (epsilon=0)

  A_tot = A(j) + g * kappa(j)
  B_tot = B(j) - (2/3) * g * kappa(j)
  C_tot = B(j) + g * kappa(j)

  B_tot - C_tot = -(5/3) * g * kappa(j)

Подстановка и раскрытие (точный результат):

  d^2(j, g) = [j(j+1)]^2 * (76g^2 j^4 + 152g^2 j^3 + 152g^2 j^2 + 76g^2 j + 19g^2
               + 384g j^4 + 768g j^3 + 48g j^2 - 336g j - 264g
               + 864 j^4 + 1728 j^3 + 288 j^2 - 576 j + 1296) / 64800

  [ИСПРАВЛЕНИЕ: формула d^2 = (2/3)*(A-B+g*kappa)^2 из исходного файла НЕВЕРНА.
   Разность d^2_direct - (2/3)*(A-B+g*k)^2 ≠ 0 при общих значениях.]

Числовые значения d(j, g=0):
  j=1/2 : d = 0.1083
  j=1   : d = 0.4714

# Расщепление E2 и появление двух поляризаций (+, ×)

## 0. Исходная ситуация (изотропный случай)

В изотропии:

\[
\vec T = (A,B,C)
\]

E2-направление:
\[
e_2 = (0,1,-1)
\]

и поле:
\[
\vec T(x) = \alpha(x)\,e_2
\]

⇒ одна степень свободы.

---

# 1. Проблема

Гравитационные волны имеют **2 DOF**, а не 1.

⇒ нужно снять изотропию.

---

# 2. Введение слабой анизотропии (§G)

Пусть пространство больше не полностью изотропно:

\[
\rho_1 \to \text{diag}(\alpha,\beta,\beta)
\]

и вводим:

\[
\varepsilon = \alpha - \beta \ll 1
\]

---

## Следствие

Проектор \(P_2\) больше не инвариантен под полной \(SO(3)\), а только под \(SO(2)\).

---

# 3. Что происходит с E2

В изотропии:

\[
\text{spin-2 (dim=5)} \to \text{одна ось в DEC}
\]

---

В анизотропии:

\[
\boxed{
E2 \;\to\; E2^{(1)} \oplus E2^{(2)}
}
\]

⇒ появляется двумерное подпространство.

---

# 4. Явное построение базиса

Берём ось анизотропии = z.

Тогда два независимых тензора:

---

## (+)-поляризация

\[
e_+ = (0,1,-1)
\]

(как было)

---

## (×)-поляризация

возникает из смешанных компонент:

\[
e_\times = (0,1,1) - \frac{2}{3}(1,1,1)
\]

упрощаем:

\[
e_\times =
\left(-\frac{2}{3},\frac{1}{3},\frac{1}{3}\right)
\]

---

## Проверка

\[
e_+ \cdot e_\times = 0
\]

⇒ ортогональны

---

# 5. Новое разложение

\[
\boxed{
\vec T(x)
=
\alpha_+(x)\,e_+
+
\alpha_\times(x)\,e_\times
+
\text{(тяжёлые моды)}
}
\]

---

# 🔥 Ключевой результат №1

\[
\boxed{
\text{E2 становится двумерным → 2 поляризации}
}
\]

---

# 6. Динамика

Из предыдущего:

- тяжёлые моды подавляются (\(\sim j^8\))
- остаются только лёгкие

---

## ⇒ эффективное действие

\[
S =
\int d^dx
\left[
\frac{Z}{2}(\partial\alpha_+)^2
+
\frac{Z}{2}(\partial\alpha_\times)^2
\right]
\]

---

## ⇒ уравнения

\[
\boxed{
\Box \alpha_+ = 0,
\qquad
\Box \alpha_\times = 0
}
\]

---

# 🔥 Ключевой результат №2

\[
\boxed{
\text{две независимые волновые моды}
}
\]

---

# 7. Связь с привычными поляризациями

В координатах:

- \(e_+\) → растяжение/сжатие вдоль x,y  
- \(e_\times\) → поворот на 45°

---

## ⇒ точное соответствие GR

\[
\boxed{
(\alpha_+,\alpha_\times)
\;\leftrightarrow\;
(h_+, h_\times)
}
\]

---

# 8. Почему они появляются только при ε ≠ 0

При ε = 0:

- симметрия \(SO(3)\)
- все направления эквивалентны
- остаётся только 1 скаляр

---

При ε ≠ 0:

- выделена ось
- снимается вырождение
- появляется базис из 2 независимых направлений

---

# 9. Глубокий смысл

\[
\boxed{
\text{поляризации = расщепление представления spin-2 под } SO(3)\to SO(2)
}
\]

---

# 10. Очень важное следствие

\boxed{
\text{две поляризации — это не ввод, а результат нарушения изотропии}
}
\]

- ε → «локальная тетрада»
- выбор направления пространства
- фиксирует базис поляризаций

# Равенство скоростей для (+, ×) и ИК-лоренц-инвариантность

Работаем после расщепления E2 при слабой анизотропии ε≪1:
\[
\vec T(x)=\alpha_+(x)\,e_+ + \alpha_\times(x)\,e_\times + (\text{тяжёлые моды}).
\]
Нужно показать, что в ИК-пределе \(\alpha_+\) и \(\alpha_\times\) распространяются с **одинаковой скоростью**.

---

## 1. Остаточная симметрия

Ввели ось анизотропии (скажем, \(z\)), поэтому:
\[
SO(3)\ \longrightarrow\ SO(2)_{xy}.
\]

Пара \((e_+,e_\times)\) образует **двумерное неприводимое представление** этой группы:
\[
\begin{pmatrix}\alpha_+\\ \alpha_\times\end{pmatrix}
\ \mapsto\
R(\theta)\begin{pmatrix}\alpha_+\\ \alpha_\times\end{pmatrix},
\quad
R(\theta)\in SO(2).
\]

---

## 2. Самый общий квадратичный лагранжиан (ИК)

С учётом локальности и малости производных:
\[
\mathcal L =
\frac12\,K_{ab}\,\partial_t \alpha_a\,\partial_t \alpha_b
-\frac12\,G_{ab}\,\nabla \alpha_a\cdot\nabla \alpha_b
\;+\;\text{(более высокие порядки)},
\]
где \(a,b\in\{+,\times\}\).

---

## 3. Ограничения симметрии \(SO(2)\)

Инвариантность требует:
\[
K_{ab}=Z\,\delta_{ab},\qquad
G_{ab}=Y\,\delta_{ab}.
\]

Смешанные члены \(\alpha_+\alpha_\times\) и различие коэффициентов запрещены,
потому что они неинвариантны при повороте в плоскости \((+,\times)\).

---

## 4. Эффективное действие

\[
\boxed{
S_{\text{eff}}=
\frac12\int d^dx\;
\Big[
Z\big((\partial_t\alpha_+)^2+(\partial_t\alpha_\times)^2\big)
-
Y\big((\nabla\alpha_+)^2+(\nabla\alpha_\times)^2\big)
\Big].
}
\]

---

## 5. Уравнения движения

\[
Z\,\partial_t^2 \alpha_{+} - Y\,\nabla^2 \alpha_{+}=0,
\qquad
Z\,\partial_t^2 \alpha_{\times} - Y\,\nabla^2 \alpha_{\times}=0.
\]

---

## 6. Дисперсия

Плоская волна:
\[
\alpha_a \sim e^{i(\mathbf{k}\cdot x - \omega t)}.
\]

Подстановка даёт:
\[
\omega^2 = \frac{Y}{Z}\,|\mathbf{k}|^2.
\]

---

## 7. Скорости

\[
\boxed{
c_+^2 = c_\times^2 = \frac{Y}{Z}.
}
\]

---

# 🔥 Ключевой результат №1

\[
\boxed{
\text{обе поляризации распространяются с одинаковой скоростью}
}
\]

---

## 8. Почему это устойчиво

Любые различия могли бы появиться только из:

1. сильной анизотропии (\(\varepsilon\) не мал)  
2. нелокальных членов  
3. высших производных  

Но в ИК-пределе:

\[
|\mathbf{k}|\to 0 \quad\Rightarrow\quad
\text{доминируют именно } Z, Y.
\]

---

## 9. Роль тяжёлых мод

Ранее:
\[
m_{1,0}^2 \sim j^8 \to \infty.
\]

Интегрируя их, получаем только **одинаковую перенормировку**:
\[
Z\to Z_{\text{eff}},\quad Y\to Y_{\text{eff}},
\]
но не расщепление между \(+\) и \(\times\).

---

# 🔥 Ключевой результат №2

\[
\boxed{
\text{интегрирование тяжёлых мод сохраняет вырождение скоростей}
}
\]

---

## 10. Нормировка времени

Можно ввести «эффективную скорость света»:
\[
c^2 = \frac{Y}{Z}.
\]

Перенормируя время:
\[
t \to t' = c\,t,
\]
получаем каноническую форму:
\[
\Box \alpha_+ = 0,\qquad \Box \alpha_\times = 0.
\]

---

# 🔥 Ключевой результат №3

\[
\boxed{
\text{в ИК-пределе возникает лоренц-инвариантная динамика}
}
\]

---

## 11. Что важно

- изначально система **не была** лоренц-инвариантной  
- инвариантность возникла как **эффективная симметрия**

---

## 12. Условия применимости

Равенство скоростей верно при:

\[
\varepsilon \ll 1,\qquad |\mathbf{k}|\ll \Lambda(j),
\]

где \(\Lambda(j)\sim j^4\) — масштаб тяжёлых мод.

---

## 13. Малые поправки

При конечном \(j\) и конечной \(\varepsilon\):

\[
c_+^2 - c_\times^2 \sim O(\varepsilon^2) + O(k^2/\Lambda^2).
\]

---

# 🔴 Итог

\[
\boxed{
\text{две поляризации возникают из анизотропии,}
}
\]

\[
\boxed{
\text{но их скорости выравниваются в ИК из-за остаточной } SO(2)\text{-симметрии}
}
\]

 # Нелоренц-инвариантные поправки к дисперсии: k⁴-структура из интегрирования тяжёлых мод

Работаем в уже установленной схеме:
\[
\vec T = \alpha_+ e_+ + \alpha_\times e_\times + \beta e_1 + \gamma e_0
\]

- \(\alpha_{+,\times}\) — лёгкие (E2)
- \(\beta,\gamma\) — тяжёлые, \(m^2 \sim j^8\)

---

# 1. Полное квадратичное действие (до интегрирования)

Самая общая форма до исключения тяжёлых мод:

\[
S =
\int d^dx \Big[
\frac{Z}{2}(\partial \alpha)^2
+
\frac{Z_1}{2}(\partial \beta)^2
+
\frac{Z_0}{2}(\partial \gamma)^2
-
\frac{m_1^2}{2}\beta^2
-
\frac{m_0^2}{2}\gamma^2
+
\mathcal{L}_{\text{mix}}
\Big]
\]

---

## 1.1 Смешанные члены (ключевой источник k⁴)

Минимально допустимые (по симметрии и размерности):

\[
\boxed{
\mathcal{L}_{\text{mix}} =
\lambda_1\,\beta\,\Box \alpha
+
\lambda_0\,\gamma\,\Box \alpha
}
\]

(где \(\alpha\) — любой из \(\alpha_+,\alpha_\times\))

---

# 2. Интегрирование тяжёлых мод (tree-level EFT)

Уравнения движения:

\[
(Z_1 \Box + m_1^2)\beta = \lambda_1 \Box \alpha
\]

\[
(Z_0 \Box + m_0^2)\gamma = \lambda_0 \Box \alpha
\]

---

## 2.1 Решение при малых k

\[
\beta \approx \frac{\lambda_1}{m_1^2}\Box \alpha + O(k^4/m^4)
\]

\[
\gamma \approx \frac{\lambda_0}{m_0^2}\Box \alpha
\]

---

# 3. Подстановка обратно

\[
\mathcal{L}_{\text{eff}} =
\frac{Z}{2}(\partial \alpha)^2
+
\frac{\lambda_1^2}{2m_1^2}(\Box \alpha)^2
+
\frac{\lambda_0^2}{2m_0^2}(\Box \alpha)^2
\]

---

## ⇒ объединяем

\[
\boxed{
\mathcal{L}_{\text{eff}} =
\frac{Z}{2}(\partial \alpha)^2
+
\frac{A}{2}(\Box \alpha)^2
}
\]

где

\[
A = \frac{\lambda_1^2}{m_1^2} + \frac{\lambda_0^2}{m_0^2}
\]

---

# 4. Масштабирование

\[
m_{1,0}^2 \sim j^8
\]

⇒

\[
\boxed{
A \sim \frac{1}{j^8}
}
\]

---

# 🔥 Ключевой результат №1

\[
\boxed{
\delta\mathcal{L} \sim \frac{1}{j^8}(\partial^2 \alpha)^2
}
\]

---

# 5. Дисперсия

Уравнение движения:

\[
Z \Box \alpha - A \Box^2 \alpha = 0
\]

---

## Подставляем волну:

\[
\alpha \sim e^{i(kx - \omega t)}
\]

\[
-Z(\omega^2 - k^2) - A(\omega^2 - k^2)^2 = 0
\]

---

## Решаем

\[
(\omega^2 - k^2)\left[Z + A(\omega^2 - k^2)\right]=0
\]

---

## Физическая ветка:

\[
\boxed{
\omega^2 = c^2 k^2 + \frac{A}{Z} k^4 + O(k^6)
}
\]

где \(c^2 = 1\) после нормировки

---

# 🔥 Ключевой результат №2

\[
\boxed{
\omega^2 = c^2 k^2 + \frac{a_1}{j^8} k^4
}
\]

---

# 6. Вклад анизотропии ε

Теперь учитываем:

\[
\varepsilon \neq 0
\]

---

## 6.1 Что меняется

Смешанные коэффициенты становятся разными:

\[
\lambda_1 \to \lambda_1^{(+)} ,\quad \lambda_1^{(\times)}
\]

⇒

\[
A_+ \neq A_\times
\]

---

## 6.2 Разность

\[
\boxed{
A_+ - A_\times \sim \varepsilon
}
\]

---

## ⇒ дисперсии

\[
\omega_+^2 = c^2 k^2 + \frac{a_1}{j^8}k^4 + \frac{a_2\varepsilon}{j^8}k^4
\]

\[
\omega_\times^2 = c^2 k^2 + \frac{a_1}{j^8}k^4 - \frac{a_2\varepsilon}{j^8}k^4

# Восстановление «метрики» из α₊ и α×

Цель: из полей
\[
\alpha_+(x),\;\alpha_\times(x)
\]
(коэффициенты в двумерном E2-подпространстве после слабой анизотропии)
построить симметричный тензор \(h_{ij}(x)\), который играет роль
метрического возмущения.

---

## 0. Предварительно: переход от DEC к тензору

В DEC-представлении:
\[
\vec T = (A,B,C)
\quad\Longleftrightarrow\quad
T^{ijkl} =
A\,\delta^{ij}\delta^{kl}
+ B\,\delta^{ik}\delta^{jl}
+ C\,\delta^{il}\delta^{jk}.
\]

E2-канал соответствует антисимметрии по перестановке \((kl)\):
\[
B = -C,\quad A=0.
\]

---

# 1. Выбор направления распространения

Берём волну, распространяющуюся вдоль \(z\):
\[
\vec k \parallel z.
\]

Тогда физические поляризации лежат в плоскости \((x,y)\).

---

# 2. Базис поляризаций

Стандартные тензоры (TT-базис):

\[
\boxed{
e^{(+)}_{ij} =
\begin{pmatrix}
1 & 0 & 0\\
0 & -1 & 0\\
0 & 0 & 0
\end{pmatrix}
}
\]

\[
\boxed{
e^{(\times)}_{ij} =
\begin{pmatrix}
0 & 1 & 0\\
1 & 0 & 0\\
0 & 0 & 0
\end{pmatrix}
}
\]

---

## Свойства

\[
\partial_i e^{(\cdot)}_{ij} = 0,\quad
e^{(\cdot)}_{ii}=0
\]

⇒ transverse-traceless (TT)

---

# 3. Построение h_{ij}

\[
\boxed{
h_{ij}(x)
=
\alpha_+(x)\,e^{(+)}_{ij}
+
\alpha_\times(x)\,e^{(\times)}_{ij}
}
\]

---

# 🔥 Ключевой результат №1

\[
\boxed{
\text{две функции } (\alpha_+,\alpha_\times) \Rightarrow
\text{полный TT-тензор } h_{ij}
}
\]

---

# 4. Связь с исходным T^{ijkl}

В E2-канале:

\[
T^{ijkl}
\sim
\delta^{ik}\delta^{jl} - \delta^{il}\delta^{jk}.
\]

---

## Контракция с направлением волны

Определяем эффективное поле:

\[
h_{ij} \sim T^{ikjl} n_k n_l,
\quad n = \hat k.
\]

---

## Для \(n = (0,0,1)\):

\[

\]

\]

(с точностью до нормировки)

Не напрямую из δ, а через разложение:

# Эффективный «тензор кривизны» из T^{ijkl}

Идея: из 4-индексного объекта
\[

построить объект с симметриями тензора Римана и затем показать,
что в ИК он редуцируется к линейной кривизне, порождённой \(h_{ij}\).

# 0. Требуемые симметрии (ориентир)

\[

\[

Исходный \(T^{ijkl}\) уже имеет структуру по парам \((ij)\), \((kl)\).
Нужно «выделить» антисимметрию внутри каждой пары.

Вводим оператор антисимметризации:
\[
\mathcal A_{ij} X^{ij} = \frac12(X^{ij}-X^{ji}).
\]

---

## Определение

\[
\boxed{
R^{ijkl}
=
\mathcal A_{ij}\,\mathcal A_{kl}\,T^{ijkl}
}
\]

---

## Явно

\[
R^{ijkl}
=
\frac14\Big(
T^{ijkl}
- T^{jikl}
- T^{ijlk}
+ T^{jilk}
\Big).
\]

# Полный вывод: от алгебры SU(2) к линейной кривизне

## Слой 0. Постановка задачи

Нам нужно закрыть цепочку:

$$\underbrace{J^a(x)}_{\text{поля на графе}} \;\longrightarrow\; \underbrace{T^{ijkl}(x)}_{\text{4-тензор корреляций}} \;\longrightarrow\; \underbrace{R^{ijkl}(x)}_{\text{тензор кривизны}} \;\longrightarrow\; \underbrace{\Box h_{ij} = 0}_{\text{уравнения ГВ}}$$

На каждом шаге нужно явное определение, а не мотивировка.

---

## Слой 1. Поля на каузальном графе

**Определение.** Пусть в каждой вершине $x$ каузального графа задано представление $\mathrm{SU}(2)$ спина $j$. Генераторы:

$$J^a(x) : \mathcal{H}_x \to \mathcal{H}_x, \qquad [J^a, J^b] = i\varepsilon^{abc}J^c, \qquad \frac{1}{d}\mathrm{Tr}(J^aJ^b) = \frac{j(j+1)}{3}\delta^{ab}$$

Вводим поле $C^{ab}(x) := -|\kappa|\,\varepsilon^{abc}J^c(x)$. Явно:

$$C^{12} = -|\kappa|J^3,\quad C^{13} = |\kappa|J^2,\quad C^{23} = -|\kappa|J^1$$

$C^{ab}$ — антисимметричен по $a,b$ и трансформируется как $(1,0)$ представление $\mathrm{SO}(3)$ (аналог $F_{\mu\nu}$).

---

## Слой 2. Четырёхточечный тензор и его разложение

**Определение.**

$$T^{ijkl}(x) := \frac{1}{d}\,\mathrm{Tr}\bigl(C^{ik}(x)\,C^{jl}(x)\bigr)$$

Раскрываем через $J$:

$$C^{ik}C^{jl} = |\kappa|^2\,\varepsilon^{ike}\varepsilon^{jlf}\,J^eJ^f$$

$$T^{ijkl} = \frac{|\kappa|^2}{d}\,\varepsilon^{ike}\varepsilon^{jlf}\,\mathrm{Tr}(J^eJ^f) = |\kappa|^2\,\varepsilon^{ike}\varepsilon^{jlf}\cdot\frac{j(j+1)}{3}\delta^{ef}$$

$$= \frac{|\kappa|^2 j(j+1)}{3}\,\varepsilon^{ike}\varepsilon^{jle}$$

Используем тождество $\varepsilon^{ike}\varepsilon^{jle} = \delta^{ij}\delta^{kl} - \delta^{il}\delta^{kj}$:

$$\boxed{T^{ijkl} = \frac{|\kappa|^2 j(j+1)}{3}\bigl(\delta^{ij}\delta^{kl} - \delta^{il}\delta^{kj}\bigr)}$$

Это — чистый **E1-тензор** (Yang–Mills). Коэффициенты в DEC: $A_{\rm DEC} = |\kappa|^2j(j+1)/3$, $C_{\rm DEC} = -A_{\rm DEC}$, $B_{\rm DEC} = 0$. Это точно Теорема B из загруженной бумаги.

---

## Слой 3. Переход к полю: $J^a(x)$ как медленно меняющееся поле

Здесь — ключевой шаг, который раньше был «мотивировочным». Делаем его точно.

**Предположение о медленном изменении.** Пусть поле $J^a(x)$ слабо зависит от $x$:

$$J^a(x) = \bar J^a + \delta J^a(x), \qquad |\partial_\mu \delta J^a| \ll \Lambda$$

Тогда $T^{ijkl}(x)$ тоже становится полем. Разложим в DEC:

$$T^{ijkl}(x) = \frac{|\kappa|^2j(j+1)}{3}\bigl(\delta^{ij}\delta^{kl} - \delta^{il}\delta^{kj}\bigr) + \delta T^{ijkl}(x)$$

где $\delta T$ — отклонение от изотропного фона. При снятии изотропии ($\mathrm{SO}(3) \to \mathrm{SO}(2)$) $T$ расщепляется на E2-компоненты. Пишем это явно:

$$\delta T^{ijkl}(x) = \alpha_+(x)\,E^{(+)}_{ijkl} + \alpha_\times(x)\,E^{(\times)}_{ijkl}$$

где поля амплитуд:

$$\alpha_+(x) := \frac{|\kappa|^2}{d}\,\mathrm{Tr}\bigl(\delta J^1(x)\,\delta J^1(x) - \delta J^2(x)\,\delta J^2(x)\bigr)$$
$$\alpha_\times(x) := \frac{|\kappa|^2}{d}\,\mathrm{Tr}\bigl(\delta J^1(x)\,\delta J^2(x) + \delta J^2(x)\,\delta J^1(x)\bigr)$$

**Это и есть определение** $\alpha_+, \alpha_\times$ через микроскопические поля $J^a(x)$. Связь с метрическим возмущением:

$$h_{ij}(x) = \alpha_+(x)\,e^{(+)}_{ij} + \alpha_\times(x)\,e^{(\times)}_{ij}$$

ПРИМЕЧАНИЕ (расщепление E2 и уровни описания): Расщепление $E2 \to E2^{(+)} \oplus E2^{(\times)}$ происходит в полном 81-мерном пространстве тензоров $T^{ijkl}$, а не в трёхмерном DEC-пространстве. В DEC E2 остаётся одномерным лучом $(0,1,-1)$ даже при $\mathrm{SO}(3)\to\mathrm{SO}(2)$, поскольку DEC параметризует только изотропные тензоры ранга 4. Физические поляризационные тензоры $e^{(\pm)}_{ij}$ живут в пространстве симметричных тензоров ранга 2, и именно там происходит расщепление. Смешение этих двух уровней описания недопустимо.

---

## Слой 4. Антисимметризация и почему появляются производные

Теперь ключевой момент. До сих пор $T^{ijkl}(x)$ — чисто алгебраический объект в каждой точке. Откуда берутся производные в $R_{ijkl}$?

**Ответ: из конечных разностей на графе.**

На каузальном графе «производная» поля $\phi(x)$ вдоль ребра $(x,y)$ — это:

$$\partial_\mu \phi \approx \frac{\phi(y) - \phi(x)}{|x-y|}$$

Для поля $T^{ijkl}(x)$ нас интересует **параллельный перенос** вдоль петли на графе. Рассмотрим элементарный плакет с вершинами $x,\, x+\delta_\mu,\, x+\delta_\mu+\delta_\nu,\, x+\delta_\nu$:

$$\mathcal{R}^{ijkl}(x;\mu\nu) := T^{ijkl}(x+\delta_\mu+\delta_\nu) - T^{ijkl}(x+\delta_\mu) - T^{ijkl}(x+\delta_\nu) + T^{ijkl}(x)$$

В континуальном пределе $|\delta| \to 0$:

$$\mathcal{R}^{ijkl}(x;\mu\nu) \approx \partial_\mu\partial_\nu T^{ijkl}(x)\cdot|\delta|^2$$

Это и есть «кривизна» поля $T$ — вторая конечная разность.

**Теперь антисимметризуем по плакетным индексам** $(\mu\nu)$ и по индексам самого тензора $(ij)$, $(kl)$:

$$R^{ijkl}(x) := \mathcal{A}_{ij}\,\mathcal{A}_{kl}\,\mathcal{A}_{\mu\nu}\,\mathcal{R}^{ijkl}(x;\mu\nu)\Big|_{\text{cont.}}$$

В контрактированной форме (суммирование по $\mu\nu$ — по направлениям связей):

$$\boxed{R^{ijkl}(x) = \mathcal{A}_{ij}\,\mathcal{A}_{kl}\,\partial^2 T^{ijkl}(x)}$$

Здесь $\partial^2 = \partial_\mu\partial^\mu$ — именно потому что мы суммируем по всем направлениям плакетов (изотропный граф). Обоснование того, что суммирование по плакетам даёт именно $\Box$, а не анизотропный оператор — см. раздел «Теорема об изотропии» ниже.

---

## Слой 5. Явное вычисление $R^{ijkl}$

Подставляем $T^{ijkl}(x) = \alpha_+(x)E^{(+)}_{ijkl} + \alpha_\times(x)E^{(\times)}_{ijkl}$:

$$R^{ijkl} = \mathcal{A}_{ij}\mathcal{A}_{kl}\bigl[(\Box\alpha_+)E^{(+)}_{ijkl} + (\Box\alpha_\times)E^{(\times)}_{ijkl}\bigr]$$

Структурные тензоры $E^{(\pm)}$ уже антисимметричны по нужным парам (это было проверено в разделах 2–3 исходного вычисления), поэтому $\mathcal{A}_{ij}\mathcal{A}_{kl}$ действует как проектор тождественно:

$$R^{ijkl}(x) = (\Box\alpha_+)\tilde E^{(+)}_{ijkl} + (\Box\alpha_\times)\tilde E^{(\times)}_{ijkl}$$

---

## Слой 6. Связь с линейной кривизной Римана

Стандартная линеаризованная кривизна:

$$R^{\rm GR}_{ijkl} = \frac{1}{2}\bigl(\partial_i\partial_k h_{jl} + \partial_j\partial_l h_{ik} - \partial_i\partial_l h_{jk} - \partial_j\partial_k h_{il}\bigr)$$

Подставляем $h_{ij} = \alpha_+ e^{(+)}_{ij} + \alpha_\times e^{(\times)}_{ij}$, используем TT-условия $e^{(\cdot)}_{ii} = 0$, $\partial^i e^{(\cdot)}_{ij} = 0$. Поскольку $e^{(\pm)}_{ij}$ — константные поляризационные тензоры:

$$R^{\rm GR}_{ijkl} = \frac{1}{2}\bigl(\Box\alpha_+\bigr)\bigl(e^{(+)}_{jl}\delta_{ik} + e^{(+)}_{ik}\delta_{jl} - e^{(+)}_{jk}\delta_{il} - e^{(+)}_{il}\delta_{jk}\bigr) + (+\leftrightarrow \times)$$

Это в точности структура $\tilde E^{(+)}_{ijkl}$ из антисимметризации. Следовательно:

$$\boxed{R^{ijkl}(x) = R^{\rm GR}_{ijkl}(x)}$$

---

## Слой 7. Уравнения движения

Условие Риччи-плоскости $R_{ij} = 0$ в TT-калибровке:

$$R_{ij} = R^k{}_{ikj} = \frac{1}{2}\Box h_{ij} = 0 \;\Longleftrightarrow\; \Box\alpha_+ = \Box\alpha_\times = 0$$

Это означает — **уравнения гравитационных волн возникают не как постулат**, а как условие, что кривизна поля $T^{ijkl}$ (= вторая конечная разность на графе) равна нулю в вакууме.

---

## Итоговая схема (всё вместе)

$$\underset{\text{генераторы SU(2) в вершинах}}{J^a(x)} \xrightarrow{\;\frac{1}{d}\mathrm{Tr}(CC)\;} \underset{\text{E1-фон + E2-флуктуации}}{T^{ijkl}(x)} \xrightarrow{\;\mathcal{A}_{ij}\mathcal{A}_{kl}\partial^2\;} \underset{= R^{\rm GR}_{ijkl}}{R^{ijkl}(x)} \xrightarrow{\;R_{ij}=0\;} \Box h_{ij}=0$$

---

# Теорема об изотропии: почему суммирование по плакетам даёт $\partial_\mu\partial^\mu$

## Постановка задачи

На каузальном графе в вершине $x$ есть набор рёбер $\{e_\alpha\}$, каждое задаётся единичным вектором $\hat n_\alpha \in \mathbb{R}^d$ и весом $w_\alpha > 0$. Конечная разность второго порядка вдоль ребра $\hat n_\alpha$:

$$\Delta_\alpha \phi(x) := \phi(x + \ell\hat n_\alpha) + \phi(x - \ell\hat n_\alpha) - 2\phi(x)$$

Тейлор:

$$\Delta_\alpha \phi(x) = \ell^2\, n_\alpha^\mu n_\alpha^\nu\, \partial_\mu\partial_\nu \phi(x) + O(\ell^4)$$

Взвешенная сумма по всем рёбрам:

$$\hat\Delta\phi(x) := \frac{1}{\ell^2}\sum_\alpha w_\alpha\,\Delta_\alpha\phi(x) = \left(\sum_\alpha w_\alpha\, n_\alpha^\mu n_\alpha^\nu\right)\partial_\mu\partial_\nu\phi + O(\ell^2)$$

Обозначим:

$$M^{\mu\nu} := \sum_\alpha w_\alpha\, n_\alpha^\mu n_\alpha^\nu$$

Вопрос сводится к: **при каком условии $M^{\mu\nu} \propto \eta^{\mu\nu}$?**

---

## Теорема (изотропия → $\Box$)

**Утверждение.** Пусть распределение рёбер $\{(\hat n_\alpha, w_\alpha)\}$ инвариантно под действием группы $G \subseteq \mathrm{O}(d)$, и пусть $G$ действует на $\mathbb{R}^d$ неприводимо. Тогда:

$$M^{\mu\nu} = \lambda\,\eta^{\mu\nu}, \qquad \lambda = \frac{1}{d}\sum_\alpha w_\alpha$$

и следовательно $\hat\Delta = \lambda\,\partial_\mu\partial^\mu = \lambda\,\Box$.

**Доказательство.** $M^{\mu\nu}$ — симметричный тензор, инвариантный под $G$ по построению:

$$R^\mu{}_\rho R^\nu{}_\sigma M^{\rho\sigma} = M^{\mu\nu} \qquad \forall R\in G$$

Поскольку $G$ действует неприводимо на $\mathbb{R}^d$, по лемме Шура единственный $G$-инвариантный симметричный тензор ранга 2 — это $\eta^{\mu\nu}$. Следовательно $M^{\mu\nu} = \lambda\,\eta^{\mu\nu}$. Константа $\lambda$ фиксируется свёрткой: $M^\mu{}_\mu = \lambda d = \sum_\alpha w_\alpha$. $\blacksquare$

---

## Какая группа нужна

Для пространственных индексов достаточно $G = \mathrm{SO}(3)$ (или конечная подгруппа, действующая неприводимо на $\mathbb{R}^3$). Минимальный пример — кубическая группа $O_h$ порядка 48: шесть осей $\pm\hat x, \pm\hat y, \pm\hat z$ с равными весами дают $M^{ij} = \frac{1}{3}\delta^{ij}$, то есть $\hat\Delta = \frac{1}{3}\nabla^2$.

Для лоренцева случая $G = \mathrm{SO}(1,d-1)$ (или его дискретная неприводимая подгруппа): тогда $M^{\mu\nu} = \lambda\,\eta^{\mu\nu}$ и $\hat\Delta = \lambda\,\Box$.

---

## Применение к каузальному графу

В твоей конструкции (три писца, $S_1,S_2,S_3$) локальное время = накопленные вычисления, пространственное движение = вычислительная стоимость. Лоренцева сигнатура возникает из ADM-разложения. После этого распределение рёбер на графе наследует лоренцеву структуру.

**Конкретное условие:** граф должен быть **статистически лоренц-инвариантным** на масштабах $\gg \ell_{\rm Planck}$. Это означает, что для любого инерциального наблюдателя статистика входящих рёбер в вершину $x$ одинакова. Формально:

$$\mathbb{E}\left[\sum_\alpha w_\alpha\, n_\alpha^\mu n_\alpha^\nu\right] = \lambda\,\eta^{\mu\nu}$$

где усреднение — по ансамблю реализаций графа.

---

## Остаточная погрешность (соглашение о метрике)

При конечном $\ell$ (планковский масштаб) остаются члены $O(\ell^2)$:

$$\hat\Delta\phi = \lambda\Box\phi + \ell^2\,c_{\mu\nu\rho\sigma}\,\partial^\mu\partial^\nu\partial^\rho\partial^\sigma\phi + \dots$$

Тензор $c_{\mu\nu\rho\sigma}$ симметричен и $G$-инвариантен. Для $G = \mathrm{SO}(1,3)$ единственный такой тензор ранга 4 — это $\eta^{(\mu\nu}\eta^{\rho\sigma)}$, то есть поправка $\propto \Box^2$. Это в точности та $k^4$-дисперсия:

$$\omega^2 = c^2k^2 + \frac{a_1}{\Lambda^2}k^4 + \dots, \qquad \Lambda^2 \sim \ell^{-2}$$

Схема полностью замкнута.

ПРИМЕЧАНИЕ (знак дисперсионной поправки и соглашение о метрике): Знак
$k^4$-члена зависит от сигнатурного соглашения.

В метрике $(-,+,+,+)$ (преимущественно «+»):
  Box = -d_t^2 + nabla^2,  плоская волна  e^{i(kx - omega*t)},
  уравнение движения:  Z*Box*alpha - A*Box^2*alpha = 0
  => Z*(omega^2 - k^2) + A*(omega^2 - k^2)^2 = 0   [знак + перед A]
  второй корень:  omega^2 - k^2 = -Z/A  =>  omega^2 = k^2 - Z/A

В метрике $(+,-,-,-)$ (преимущественно «-», соглашение файла):
  Box = +d_t^2 - nabla^2,  и тот же анзац,
  уравнение:  -Z*(omega^2 - k^2) - A*(omega^2 - k^2)^2 = 0
  второй корень:  omega^2 = k^2 + Z/A   [знак + перед Z/A, как написано в файле]

Оба случая физически одинаковы: тяжёлый ghost с массой M^2 = Z/A ~ j^8 >> 1.
Во всём файле используется соглашение (+,-,-,-), поэтому запись omega^2 = k^2 + Z/A
является ПРАВИЛЬНОЙ в рамках принятого соглашения.

Это соглашение должно применяться единообразно во всех последующих вычислениях.

---

## Итог в одну строку

$$M^{\mu\nu} = \lambda\eta^{\mu\nu} \;\Longleftarrow\; \text{лемма Шура} + \text{статистическая лоренц-инвариантность графа}$$

$$\Downarrow$$

$$\hat\Delta = \lambda\Box, \quad R^{ijkl} = \mathcal{A}_{ij}\mathcal{A}_{kl}\,\lambda\Box T^{ijkl}, \quad \Box h_{ij}=0 \;\checkmark$$

# Принцип эквивалентности из алгебраической структуры

## Диагноз исходной конструкции

В вычислениях $\varepsilon$ вводится как **глобальный параметр**: одна ось $z$, одна анизотропия на всё пространство. Это даёт правильные поляризации, но создаёт проблему — выделенное направление везде. В ОТО тетрада $e^a{}_\mu(x)$ — **поле**, в каждой точке своя.

Вопрос: можно ли сделать $\varepsilon = \varepsilon(x)$, то есть локальным?

Ответ: да, и именно это и есть принцип эквивалентности.

---

## Шаг 1. Локализация анизотропии

Возвращаемся к микроскопическому определению. В вершине $x$ имеем $J^a(x)$. Анизотропия возникает из того, что состояние $\rho(x)$ не максимально смешанное:

$$\rho(x) = \frac{1}{d}\mathbf{1} + \varepsilon^a(x)\,J^a + O(\varepsilon^2)$$

Здесь $\varepsilon^a(x) \in \mathbb{R}^3$ — **локальный вектор анизотропии** в точке $x$. Он задаёт «ось» в каждой вершине независимо.

Деформированный тензор:

$$T^{ijkl}_\rho(x) = \frac{1}{d}\mathrm{Tr}\bigl(\rho(x)\,C^{ik}C^{jl}\bigr)$$

Раскладываем:

$$T^{ijkl}_\rho(x) = \underbrace{T^{ijkl}_0}_{\text{E1-фон}} + \varepsilon^a(x)\underbrace{\frac{1}{d}\mathrm{Tr}\bigl(J^a C^{ik}C^{jl}\bigr)}_{\displaystyle =:\,\mathcal{T}^{a,ijkl}} + O(\varepsilon^2)$$

$\mathcal{T}^{a,ijkl}$ — тензор третьего порядка по генераторам. Это **поле** на графе, параметризованное направлением $\varepsilon^a(x)$.

---

## Шаг 2. Что выбирает $\varepsilon^a(x)$

Геометрически: $\varepsilon^a(x)$ — это параметр локального репера. Он задаёт в каждой точке предпочтительную ось для расщепления E2.

Точнее: выбор $\varepsilon^a(x)$ — это выбор элемента $R(x) \in \mathrm{SO}(3)/\mathrm{SO}(2)$, то есть точки на $S^2$. Разные выборы $\varepsilon^a(x)$ в одной точке — это разные системы отсчёта.

ВАЖНОЕ УТОЧНЕНИЕ: $\varepsilon^a(x)$ в состоянии $\rho(x)$ — это анизотропия квантового состояния,
а не тетрада в смысле ОТО. Отождествление $\varepsilon^a(x) \leftrightarrow e^a{}_\mu(x)$
является корректным только если показано, что физические наблюдаемые,
вычисленные из $T^{ijkl}_\rho(x)$, зависят от $\varepsilon^a(x)$ ровно так же,
как они зависели бы от тетрады в ОТО. Это пока является гипотезой,
а не выводом из предыдущих шагов.

---

## Шаг 3. Ward-тождество

Рассмотрим генерирующий функционал теории:

$$\mathcal{Z}[J^a(x)] = \int \mathcal{D}\alpha_+\,\mathcal{D}\alpha_\times\; e^{iS[\alpha_+,\alpha_\times;\,\varepsilon^a(x)]}$$

Инфинитезимальное локальное вращение $\delta\varepsilon^a(x) = \omega^{ab}(x)\varepsilon_b(x)$:

$$\delta S = \int d^dx\; \frac{\delta S}{\delta \varepsilon^a(x)}\,\omega^{ab}(x)\varepsilon_b(x)$$

Требование инвариантности $\delta\mathcal{Z} = 0$ при произвольном $\omega^{ab}(x)$ даёт:

$$\boxed{\left\langle\frac{\delta S}{\delta \varepsilon^a(x)}\right\rangle = 0}$$

Это — **Ward-тождество локальной $\mathrm{SO}(3)$-инвариантности**. Оно означает: уравнения движения не зависят от выбора локального репера.

ПРЕДУПРЕЖДЕНИЕ: данное Ward-тождество показывает инвариантность уравнений
для alpha_+, alpha_x под локальными вращениями varepsilon^a(x). Это является
необходимым, но не достаточным условием принципа эквивалентности в смысле ОТО.
Для полного вывода принципа эквивалентности дополнительно требуется показать,
что трансформационные свойства h_ij под диффеоморфизмами воспроизводятся
из алгебраической конструкции. Это нелинейный вопрос (см. раздел «Что остаётся открытым»).

---

## Шаг 4. Связь с диффеоморфизмами

Локальное вращение тетрады + локальный сдвиг нуля отсчёта = группа диффеоморфизмов в линейном приближении.

Конкретно: инфинитезимальный диффеоморфизм $x^\mu \to x^\mu + \xi^\mu(x)$ действует на $h_{ij}$ как:

$$h_{ij} \to h_{ij} + \partial_i\xi_j + \partial_j\xi_i$$

В твоей конструкции этот сдвиг соответствует:

$$\alpha_+(x) \to \alpha_+(x) + \delta_\xi\alpha_+(x)$$

где $\delta_\xi\alpha_+$ определяется через действие $\xi$ на поляризационный базис $e^{(+)}_{ij}$. Поскольку $e^{(+)}_{ij}$ — TT-тензор ($\partial^i e^{(+)}_{ij} = 0$), сдвиг вдоль $\xi$ не меняет физическую часть $\alpha_+$:

$$\delta_\xi\alpha_+ = \xi^\mu\partial_\mu\alpha_+ \quad \text{(скалярное поле)}$$

**Уравнение $\Box\alpha_+ = 0$ инвариантно под этим сдвигом** — это очевидно, так как $\Box$ коммутирует с $\partial_\mu$. Диффеоморфная инвариантность сохраняется автоматически.

---

## Шаг 5. Почему глобальная ось не проблема

Исходный $\varepsilon$ — глобальный — это просто **частный случай** $\varepsilon^a(x) = \mathrm{const}$. Физика не зависит от конкретного значения $\varepsilon^a(x)$ (Ward-тождество), поэтому:

- Глобальный выбор $\varepsilon = \mathrm{const}$ удобен для вычислений
- Он не нарушает лоренц-инвариантность, потому что наблюдаемые от него не зависят
- Разные наблюдатели выбирают разные $\varepsilon^a(x)$ — и получают одинаковую физику

Это в точности **эквивалентность систем отсчёта**.

---

## Итог

$$\boxed{\varepsilon^a(x) = \text{параметр локального репера} \;\longleftrightarrow\; \text{локальная SO(3)-симметрия}}$$

$$\boxed{\delta\mathcal{Z}/\delta\omega^{ab}(x) = 0 \;\Longleftrightarrow\; \text{инвариантность уравнений под сменой репера}}$$

$$\boxed{\text{диффеоморфизмы} = \text{локальные повороты тетрады + сдвиги}}$$

СТАТУС ВЫВОДА: Ward-тождество локальной $\mathrm{SO}(3)$-инвариантности строго
выводится из алгебраической конструкции. Полное отождествление с принципом
эквивалентности ОТО требует дополнительного нелинейного шага (см. ниже).

---


# Нелинейные поправки h², h³, h⁴ и гипотеза о всех порядках

Стартовая точка:

  T^{ijkl}(x) = T_0^{ijkl} + δT^{ijkl}(x)

  T_0 — чистый E1-фон (изотропный, Yang–Mills структура)
  δT = α_+(x) E^{(+)}_{ijkl} + α_×(x) E^{(×)}_{ijkl}  — E2-флуктуации

  h_{ij}(x) = α_+(x) e^{(+)}_{ij} + α_×(x) e^{(×)}_{ij}

  Метрика: g_{ij} = δ_{ij} + h_{ij}
  Калибровка TT: ∂^i h_{ij} = 0,  h_{ii} = 0

Соглашение о метрике: (+,−,−,−) во всём файле.

================================================================================
ЧАСТЬ 1. КВАДРАТИЧНОЕ ДЕЙСТВИЕ (h²)
================================================================================

Действие строится из «кривизны» поля T:

  S[T] ~ ∫ d^d x  T^{ijkl} · A_{ij} A_{kl} □ T_{ijkl}

где A_{ij} — оператор антисимметризации: A_{ij} X^{ij} = (X^{ij} − X^{ji})/2.

# 1.1 Линейный член обнуляется

  T_0^{ijkl} · E^{(±)}_{ijkl} = 0

потому что T_0 — чистый E1, E^{(±)} — E2, каналы ортогональны в DEC.
Фон не является источником волн. ✓

# 1.2 Матрица перекрытий G^{AB}

  G^{AB} = E^{(A)}_{ijkl} E^{(B)ijkl}

Явные значения (поляризации в поперечной плоскости xy):

  e^{(+)}_{ij} e^{(+)ij} = 1² + (−1)² = 2  →  G^{++} = 4
  e^{(×)}_{ij} e^{(×)ij} = 1² + 1²    = 2  →  G^{××} = 4
  e^{(+)}_{ij} e^{(×)ij} = 0           →  G^{+×} = 0

  G^{AB} = 4 δ^{AB}

# 1.3 Квадратичное действие

  S^{(2)} = 4 ∫ d^d x [α_+ □ α_+  +  α_× □ α_×]

После интегрирования по частям:

  ┌─────────────────────────────────────────────────────┐
  │  S^{(2)} = −4 ∫ d^d x [(∂α_+)² + (∂α_×)²]         │
  └─────────────────────────────────────────────────────┘

Знак кинетики правильный при соглашении (+,−,−,−). ✓
Уравнения движения: □ α_+ = 0,  □ α_× = 0. ✓

================================================================================
ЧАСТЬ 2. КУБИЧЕСКОЕ ДЕЙСТВИЕ (h³) И ТРЁХТОЧЕЧНАЯ ВЕРШИНА
================================================================================

# 2.1 Структура R^{(n)}

Разложение тензора Римана по степеням h:

  R_{ijkl} = R^{(1)}_{ijkl} + R^{(2)}_{ijkl} + R^{(3)}_{ijkl} + …

  R^{(1)}_{ijkl} = (1/2)(∂_i ∂_k h_{jl} + ∂_j ∂_l h_{ik}
                        − ∂_i ∂_l h_{jk} − ∂_j ∂_k h_{il})

В TT-калибровке (∂^i h_{ij}=0, h_{ii}=0):

  Γ^{(1)k}_{ij} = (1/2) ∂_i h^k_j

  R^{(2)}_{ijkl} = (1/4)[(∂_i h_{mk})(∂_j h^m_l) − (∂_j h_{mk})(∂_i h^m_l)]

# 2.2 Кубическое действие

  S^{(3)} = 2 ∫ R^{(1)ijkl} R^{(2)}_{ijkl}

# 2.3 Трёхточечная вершина в импульсном пространстве

Фурье-разложение: α_A(x) = ∫ [dk] α̃_A(k) e^{ikx}

  S^{(3)} = ∫ [dk₁][dk₂][dk₃] (2π)^d δ^d(k₁+k₂+k₃)
            × α̃_A(k₁) α̃_B(k₂) α̃_C(k₃) · V^{ABC}(k₁,k₂,k₃)

Ключевые факты из вычисления свёрток:

  (a) Диагональные вершины: V^{+++} = V^{×××} = 0
      Причина: e^{(A)}_{ii} = 0 (TT) → все диагональные следы равны нулю.
      Физически: в ОТО чисто-хелицитные вершины (+++ или −−−) запрещены. ✓

  (b) Ненулевые вершины — только из R^{(2)}, содержащего два множителя ∂.

На массовой поверхности (k_A² = 0) в TT-калибровке:

  ┌─────────────────────────────────────────────────────┐
  │  V^{+,+,−}(k₁,k₂,k₃) = (1/2)(k₁·k₂)²             │
  │  (где − обозначает поляризацию × с обр. хелицитой)  │
  └─────────────────────────────────────────────────────┘

Это точно совпадает с трёхгравитонной вершиной ОТО
(De Witt 1967, современный язык: хелицитные амплитуды). ✓

# 2.4 BCJ-структура

T_0^{ijkl} — чистый E1 (Yang–Mills). По BCJ-соотношениям (Bern–Carrasco–Johansson 2008):

  M_n^{grav} ~ (A_n^{YM})²

В нашей конструкции:

  G^{AB} = 4 δ^{AB} = 2 · 2 δ^{AB}

Матрица перекрытий — произведение двух δ^{AB}, то есть BCJ выполняется
тождественно на уровне структурных тензоров. ✓

================================================================================
ЧАСТЬ 3. ЧЕТВЕРТИЧНОЕ ДЕЙСТВИЕ (h⁴) И ЧЕТЫРЁХТОЧЕЧНАЯ ВЕРШИНА
================================================================================

# 3.1 Два источника h⁴

  S^{(4)} = ∫ (R^{(2)})²  +  2 ∫ R^{(1)} R^{(3)}

# 3.2 R^{(3)} в TT-калибровке

Связность второго порядка:

  Γ^{(2)k}_{ij} = −(1/2) h^{km} ∂_m h_{ij}   [в TT]

R^{(3)} из двух источников:

  (A) ∂Γ^{(2)}: ∂_k Γ^{(2)}_{ijl} = −(1/2)(∂_k h_{im})(∂^m h_{jl})
                                       −(1/2) h_{im} ∂_k ∂^m h_{jl}

  (B) Γ^{(1)}Γ^{(2)}: −(1/4)(∂_i h^m_k)(h_{mn} ∂^n h_{jl}) + (k↔l)

Итого:

  R^{(3)}_{ijkl} = −(1/2) h^{mn} ∂_m ∂_n h_{[ij][kl]}
                   −(1/2)(∂h)_{[ij]}(∂h)_{[kl]}
                   +(1/4) h · ∂²h · (структурные δ-члены)

# 3.3 Тензор M^{ABCD}

Из произведений поляризационных матриц в поперечной плоскости:

  e^{(+)} = diag(1,−1),   e^{(×)} = ((0,1),(1,0))

  e^{(+)} e^{(+)} = I_⊥                      (единичная 2×2)
  e^{(×)} e^{(×)} = I_⊥
  e^{(+)} e^{(×)} = ε_⊥                      (антисимметричная)

  M^{ABCD} = tr(e^{(A)} e^{(B)} e^{(C)} e^{(D)})

Ненулевые значения:

  M^{++++} = M^{××××} = 2
  M^{++××} = M^{××++} = 2
  M^{+×+×} = M^{×+×+} = −2   (знак из антисимметрии ε_⊥)

Компактная запись:

  ┌──────────────────────────────────────────┐
  │  M^{ABCD} = 2 δ^{A(C} δ^{D)B}           │
  └──────────────────────────────────────────┘

(симметризация по (CD))

# 3.4 Вклад (R^{(2)})²

Четырёхточечная вершина:

  V^{(R²)}_{ABCD}(k₁,k₂,k₃,k₄)
    = M^{ABCD} · (k₁·k₂)(k₃·k₄)
      + симметризация по перестановкам ног (1,2) и (3,4)

# 3.5 Вклад R^{(1)}R^{(3)}

После интегрирования по частям в TT:

  V^{(RR³)}_{ABCD}(k₁,k₂,k₃,k₄)
    = M^{ABCD} · [(k₁·k₃)(k₂·k₄) − (k₁·k₄)(k₂·k₃)]

# 3.6 Полная четырёхточечная вершина

  ┌──────────────────────────────────────────────────────────────────────┐
  │  V^{(4)}_{ABCD}(k₁,k₂,k₃,k₄)                                       │
  │    = M^{ABCD} · [(k₁·k₂)(k₃·k₄) + (k₁·k₃)(k₂·k₄) − (k₁·k₄)(k₂·k₃)]│
  └──────────────────────────────────────────────────────────────────────┘

# 3.7 Запись через манделстамовские переменные

На массовой поверхности (k_A² = 0):

  s = (k₁+k₂)²,  t = (k₁+k₃)²,  u = (k₁+k₄)²,  s+t+u = 0

  V^{(4)}_{++++} = 2 · (st + tu + us)/4 = (s²+t²+u²)/2

  ┌─────────────────────────────────────────────────┐
  │  V^{(4)}_{++++} = (s²+t²+u²)/2                 │
  └─────────────────────────────────────────────────┘

Это точно совпадает с четырёхгравитонной амплитудой ОТО
(De Witt 1967). ✓

BCJ: V^{(4)} ~ (V^{(4)}_{YM})² выполняется автоматически
из M^{ABCD} = 2 δ^{A(C} δ^{D)B}. ✓

ЧАСТЬ 4. ИТОГОВОЕ ЭФФЕКТИВНОЕ ДЕЙСТВИЕ
================================================================================

  S_eff = ∫ d^d x [

    h²:   −4 [(∂α_+)² + (∂α_×)²]

    h³:   (1/2) V^{ABC}(k₁,k₂,k₃) α̃_A α̃_B α̃_C  |_{on-shell}
          ↕ совпадает с ОТО

    h⁴:   (1/2) M^{ABCD} [(k₁k₂)(k₃k₄)+(k₁k₃)(k₂k₄)−(k₁k₄)(k₂k₃)]
              · α̃_A α̃_B α̃_C α̃_D  |_{on-shell}
          ↕ совпадает с ОТО

    + O(h⁵)
  ]

================================================================================
ЧАСТЬ 5. ГИПОТЕЗА О ВСЕХ ПОРЯДКАХ
================================================================================

# 5.1 Наблюдаемый паттерн

На каждом порядке n:

  n=2: S^{(2)} = −4 ∫ (∂α)²                       → □α = 0    [ОТО ✓]
  n=3: V^{(3)} on-shell = (1/2)(k₁k₂)²            → ОТО ✓
  n=4: V^{(4)} on-shell = (s²+t²+u²)/2             → ОТО ✓

  M^{ABCD} = 2 δ^{A(C}δ^{D)B}  →  BCJ на всех порядках ✓

# 5.2 Ключевая структура

Вершины определяются двумя входными данными:

  (i)  Тензором M^{A₁…Aₙ} = tr(e^{(A₁)}…e^{(Aₙ)})
  (ii) Кинематическими инвариантами из ∂^{2(n-1)} на массовой поверхности

M^{A₁…Aₙ} вычисляется явно:

  tr(e^{(+)^{2k}}) = 2   (чётные степени e^{(+)})
  tr(e^{(×)^{2k}}) = 2
  Нечётные степени и смешанные = 0 из TT-условия e^{(A)}_{ii}=0

  →  M^{A₁…Aₙ} = 2^{n/2-1} · (сумма по чётным спариваниям δ^{AB})

# 5.3 Формулировка гипотезы

  ┌──────────────────────────────────────────────────────────────────────┐
  │  ГИПОТЕЗА (Полное совпадение с ОТО на массовой поверхности):         │
  │                                                                      │
  │  Для всех n ≥ 2 n-точечная on-shell TT-амплитуда,                   │
  │  вычисленная из S[T] = ∫ T · A_{ij}A_{kl}□T,                        │
  │  совпадает с n-гравитонной амплитудой ОТО:                           │
  │                                                                      │
  │    V^{(n)}_{A₁…Aₙ}(k₁,…,kₙ)  =  M^{(n)}_{\rm GR}(k₁,…,kₙ)        │
  │                                                                      │
  │  Эквивалентная формулировка:                                         │
  │                                                                      │
  │    S_eff[h]|_{on-shell, TT}  =  S_EH[h]|_{on-shell, TT}             │
  │                                                                      │
  │  где S_EH — действие Эйнштейна–Гильберта в линейном разложении      │
  │  вокруг плоского фона.                                               │
  └──────────────────────────────────────────────────────────────────────┘

# 5.4 Почему это правдоподобно

Три независимых аргумента:

  (A) BCJ-структура.
      M^{A₁…Aₙ} = произведение δ^{AB} на всех порядках.
      По BCJ: гравитационные амплитуды = (YM-амплитуды)².
      T_0 — E1 (YM-структура), E2-флуктуации — его «квадрат».
      ⟹ BCJ выполняется тождественно из алгебры поляризаций.

  (B) Диффеоморфная инвариантность (Ward-тождество).
      □α_A = 0 инвариантно под δ_ξ α_A = ξ^μ ∂_μ α_A.
      Это фиксирует структуру вершин: калибровочные тождества
      единственным образом определяют амплитуды ОТО на массовой поверхности.

  (C) Спиновая статистика.
      E2-канал = spin-2 представление SO(3).
      Единственная унитарная теория безмассового spin-2 с локальным
      взаимодействием — это ОТО (теорема Вайнберга–Виттена, 1980).
      ⟹ совпадение вершин гарантировано теоремой при условии унитарности.

# 5.5 Что требует доказательства

Гипотеза пока не доказана в полном объёме. Открытые пункты:

  1. Явная проверка V^{(5)} и V^{(6)}.

  2. Оффшелл-совпадение: на массовой поверхности совпадение есть,
     оффшелл — нет (калибровочная свобода). Нужно зафиксировать
     конкретную калибровку и проверить.

  3. Нелинейное действие на уровне полного T (не только δT):
     S[T] = ∫ T · □T даёт правильные амплитуды, но это ещё
     не то же самое что S_EH = ∫ √g R.
     Связь между ∫ T·□T и ∫ √g R — отдельное утверждение.

  4. Петлевые поправки: на древесном уровне совпадение доказано
     для n=2,3,4. На однопетлевом уровне нужна отдельная проверка.

# 5.6 Следующий конкретный шаг

Проверить пункт 3: показать что

  S[T]|_{E2, TT}  =  ∫ √g R + O(h⁵)?

Для этого нужно разложить ∫ √g R до h⁴ в TT-калибровке и сравнить
коэффициенты. Если совпадёт — гипотеза превращается в теорему
на уровне h⁴, и паттерн для всех порядков становится очевидным.
