# Нелоренц-инвариантные поправки к дисперсии: k⁴-структура из интегрирования тяжёлых мод

Работаем в уже установленной схеме:
\[
\vec T = \alpha_+ e_+ + \alpha_\times e_\times + \beta e_1 + \gamma e_0
\]

- \(\alpha_{+,\times}\) — лёгкие (E2)
- \(\beta,\gamma\) — тяжёлые, \(m^2 \sim j^8\)

---

# 1. Полное квадратичное действие (до интегрирования)

Самая общая форма до исключения тяжёлых мод:

\[
S =
\int d^dx \Big[
\frac{Z}{2}(\partial \alpha)^2
+
\frac{Z_1}{2}(\partial \beta)^2
+
\frac{Z_0}{2}(\partial \gamma)^2
-
\frac{m_1^2}{2}\beta^2
-
\frac{m_0^2}{2}\gamma^2
+
\mathcal{L}_{\text{mix}}
\Big]
\]

---

## 1.1 Смешанные члены (ключевой источник k⁴)

Минимально допустимые (по симметрии и размерности):

\[
\boxed{
\mathcal{L}_{\text{mix}} =
\lambda_1\,\beta\,\Box \alpha
+
\lambda_0\,\gamma\,\Box \alpha
}
\]

(где \(\alpha\) — любой из \(\alpha_+,\alpha_\times\))

---

# 2. Интегрирование тяжёлых мод (tree-level EFT)

Уравнения движения:

\[
(Z_1 \Box + m_1^2)\beta = \lambda_1 \Box \alpha
\]

\[
(Z_0 \Box + m_0^2)\gamma = \lambda_0 \Box \alpha
\]

---

## 2.1 Решение при малых k

\[
\beta \approx \frac{\lambda_1}{m_1^2}\Box \alpha + O(k^4/m^4)
\]

\[
\gamma \approx \frac{\lambda_0}{m_0^2}\Box \alpha
\]

---

# 3. Подстановка обратно

\[
\mathcal{L}_{\text{eff}} =
\frac{Z}{2}(\partial \alpha)^2
+
\frac{\lambda_1^2}{2m_1^2}(\Box \alpha)^2
+
\frac{\lambda_0^2}{2m_0^2}(\Box \alpha)^2
\]

---

## ⇒ объединяем

\[
\boxed{
\mathcal{L}_{\text{eff}} =
\frac{Z}{2}(\partial \alpha)^2
+
\frac{A}{2}(\Box \alpha)^2
}
\]

где

\[
A = \frac{\lambda_1^2}{m_1^2} + \frac{\lambda_0^2}{m_0^2}
\]

---

# 4. Масштабирование

\[
m_{1,0}^2 \sim j^8
\]

⇒

\[
\boxed{
A \sim \frac{1}{j^8}
}
\]

---

# 🔥 Ключевой результат №1

\[
\boxed{
\delta\mathcal{L} \sim \frac{1}{j^8}(\partial^2 \alpha)^2
}
\]

---

# 5. Дисперсия

Уравнение движения:

\[
Z \Box \alpha - A \Box^2 \alpha = 0
\]

---

## Подставляем волну:

\[
\alpha \sim e^{i(kx - \omega t)}
\]

\[
-Z(\omega^2 - k^2) - A(\omega^2 - k^2)^2 = 0
\]

---

## Решаем

\[
(\omega^2 - k^2)\left[Z + A(\omega^2 - k^2)\right]=0
\]

---

## Физическая ветка:

\[
\boxed{
\omega^2 = c^2 k^2 + \frac{A}{Z} k^4 + O(k^6)
}
\]

где \(c^2 = 1\) после нормировки

---

# 🔥 Ключевой результат №2

\[
\boxed{
\omega^2 = c^2 k^2 + \frac{a_1}{j^8} k^4
}
\]

---

# 6. Вклад анизотропии ε

Теперь учитываем:

\[
\varepsilon \neq 0
\]

---

## 6.1 Что меняется

Смешанные коэффициенты становятся разными:

\[
\lambda_1 \to \lambda_1^{(+)} ,\quad \lambda_1^{(\times)}
\]

⇒

\[
A_+ \neq A_\times
\]

---

## 6.2 Разность

\[
\boxed{
A_+ - A_\times \sim \varepsilon
}
\]

---

## ⇒ дисперсии

\[
\omega_+^2 = c^2 k^2 + \frac{a_1}{j^8}k^4 + \frac{a_2\varepsilon}{j^8}k^4
\]

\[
\omega_\times^2 = c^2 k^2 + \frac{a_1}{j^8}k^4 - \frac{a_2\varepsilon}{j^8}k^4
\]

---

# 🔥 Ключевой результат №3

\[
\boxed{
c_+^2 - c_\times^2 \sim \frac{\varepsilon}{j^8} k^2
}
\]

---

# 7. Переписываем через Λ

\[
\Lambda^2 \sim j^8
\]

---

## Итог:

\[
\boxed{
\omega^2 =
c^2 k^2
+
\frac{a_1}{\Lambda^2}k^4
+
\frac{a_2\varepsilon}{\Lambda^2}k^4
+ \dots
}
\]

---

# 🔥 Главный вывод

\[
\boxed{
\text{нарушение лоренц-инвариантности подавлено как } \frac{k^2}{\Lambda^2}
}
\]

---

# 8. Неочевидная структура

Поправка зависит от:

\[
\Box^2 \sim (\omega^2 - k^2)^2
\]

---

## ⇒

не просто \(k^4\), а **геометрически согласованная деформация**

---

# 9. Физическая интерпретация

- тяжёлые моды = «скрытая структура пространства»
- интегрирование → «жёсткость среды»
- \(k^4\) → дисперсия среды, не вакуума

---

# 🔴 Итог

\[
\boxed{
\text{в ИК — точная лоренц-инвариантность}
}
\]

\[
\boxed{
\text{в УФ — контролируемое нарушение } (\sim k^4)
}
\]

# Знак A и условия отсутствия ghost-мод

Рассматриваем эффективный лагранжиан после интегрирования тяжёлых мод:
\[
\mathcal L_{\text{eff}}
=
\frac{Z}{2}(\partial \alpha)^2
+
\frac{A}{2}(\Box \alpha)^2
\]

где:
\[
A = \frac{\lambda_1^2}{m_1^2} + \frac{\lambda_0^2}{m_0^2}
\]

---

# 1. Знак A

Очевидно:

- \(m_{1,0}^2 > 0\) (массы тяжёлых мод)
- \(\lambda_{0,1}^2 \ge 0\)

---

## ⇒

\[
\boxed{
A > 0
}
\]

---

# 🔥 Ключевой результат №1

\[
\boxed{
k^4\text{-поправка всегда положительна}
}
\]

---

# 2. Дисперсия

\[
\omega^2 = c^2 k^2 + \frac{A}{Z}k^4
\]

при \(A>0\):

- фазовая скорость растёт с k  
- группа:
\[
v_g = \frac{d\omega}{dk} > c
\]

---

## ⚠️ замечание

это **не нарушение каузальности**, потому что:

- это EFT
- при \(k \sim \Lambda\) теория перестаёт быть валидной

---

# 3. Ghost-проблема (ключевой момент)

Лагранжиан содержит:

\[
(\Box \alpha)^2
\]

⇒ уравнение 4-го порядка

---

## Это обычно ⇒ Ostrogradsky ghost

---

# 4. Проверка явно

Дисперсионное уравнение:

\[
Z(\omega^2 - k^2) - A(\omega^2 - k^2)^2 = 0
\]

---

## Корни:

\[
\omega^2 - k^2 = 0
\]

\[
\omega^2 - k^2 = \frac{Z}{A}
\]

---

## ⇒

\[
\boxed{
\omega^2 = k^2 \quad (\text{физическая мода})
}
\]

\[
\boxed{
\omega^2 = k^2 + M^2,\quad M^2 = \frac{Z}{A}
}
\]

---

# 5. Скаляризация лагранжиана

Вводим вспомогательное поле \(\chi\):

\[
\boxed{
\mathcal L =
\frac{Z}{2}(\partial \alpha)^2
-
\frac{1}{2}M^2 \chi^2
+
\chi\,\Box \alpha
}
\]

---

## Интегрируя χ → получаем обратно (\Box α)²

---

# 6. Диагонализация

После преобразования:

\[
\mathcal L \sim
\frac{Z}{2}(\partial \alpha)^2
-
\frac{1}{2}(\partial \chi)^2
-
\frac{1}{2}M^2 \chi^2
\]

---

# 🔴 Ключевой результат №2

\[
\boxed{
\chi \text{ имеет отрицательный кинетический член}
}
\]

---

## ⇒

\[
\boxed{
\text{есть ghost-мода}
}
\]

---

# 7. Но есть нюанс (очень важный)

Масса ghost:

\[
M^2 = \frac{Z}{A} \sim j^8
\]

---

## ⇒

\[
\boxed{
M \sim j^4 \to \infty
}
\]

---

# 🔥 Ключевой результат №3

\[
\boxed{
\text{ghost уходит на ультрафиолет}
}
\]

---

# 8. EFT-интерпретация

На энергиях:

\[
E \ll M \sim j^4
\]

---

## ghost:

- не возбуждается  
- не участвует в динамике  
- не нарушает унитарность

---

# 🔥 Ключевой вывод

\[
\boxed{
\text{теория без ghost в ИК, несмотря на их формальное наличие}
}
\]

---

# 9. Условия согласованности

## 1. Z > 0

⇒ правильный знак кинетики

---

## 2. A > 0

⇒ нет тахионов

---

## 3. E ≪ M

⇒ ghost не активируется

---

# 🔴 Итоговые условия

\[
\boxed{
Z > 0,\quad A > 0,\quad E \ll j^4
}
\]

---

# 10. Более глубокая интерпретация

Ghost возникает потому что:

\[
(\Box \alpha)^2
\]

— это эффективный оператор, а не фундаментальный.

---

## ⇒

\[
\boxed{
\text{ghost = артефакт усечения EFT}
}
\]

---

# 11. Связь с исходной моделью

В полной теории:

- есть только \(\alpha,\beta,\gamma\)
- нет higher-derivatives

---

## ⇒

\[
\boxed{
\text{полная теория ghost-free}
}
\]

---

# 12. Очень важный вывод

\[
\boxed{
\text{гравитационная мода стабильна в ИК}
}
\]

---

# Восстановление «метрики» из α₊ и α×

Цель: из полей
\[
\alpha_+(x),\;\alpha_\times(x)
\]
(коэффициенты в двумерном E2-подпространстве после слабой анизотропии)
построить симметричный тензор \(h_{ij}(x)\), который играет роль
метрического возмущения.

---

## 0. Предварительно: переход от DEC к тензору

В DEC-представлении:
\[
\vec T = (A,B,C)
\quad\Longleftrightarrow\quad
T^{ijkl} =
A\,\delta^{ij}\delta^{kl}
+ B\,\delta^{ik}\delta^{jl}
+ C\,\delta^{il}\delta^{jk}.
\]

E2-канал соответствует антисимметрии по перестановке \((kl)\):
\[
B = -C,\quad A=0.
\]

---

# 1. Выбор направления распространения

Берём волну, распространяющуюся вдоль \(z\):
\[
\vec k \parallel z.
\]

Тогда физические поляризации лежат в плоскости \((x,y)\).

---

# 2. Базис поляризаций

Стандартные тензоры (TT-базис):

\[
\boxed{
e^{(+)}_{ij} =
\begin{pmatrix}
1 & 0 & 0\\
0 & -1 & 0\\
0 & 0 & 0
\end{pmatrix}
}
\]

\[
\boxed{
e^{(\times)}_{ij} =
\begin{pmatrix}
0 & 1 & 0\\
1 & 0 & 0\\
0 & 0 & 0
\end{pmatrix}
}
\]

---

## Свойства

\[
\partial_i e^{(\cdot)}_{ij} = 0,\quad
e^{(\cdot)}_{ii}=0
\]

⇒ transverse-traceless (TT)

---

# 3. Построение h_{ij}

\[
\boxed{
h_{ij}(x)
=
\alpha_+(x)\,e^{(+)}_{ij}
+
\alpha_\times(x)\,e^{(\times)}_{ij}
}
\]

---

# 🔥 Ключевой результат №1

\[
\boxed{
\text{две функции } (\alpha_+,\alpha_\times) \Rightarrow
\text{полный TT-тензор } h_{ij}
}
\]

---

# 4. Связь с исходным T^{ijkl}

В E2-канале:

\[
T^{ijkl}
\sim
\delta^{ik}\delta^{jl} - \delta^{il}\delta^{jk}.
\]

---

## Контракция с направлением волны

Определяем эффективное поле:

\[
h_{ij} \sim T^{ikjl} n_k n_l,
\quad n = \hat k.
\]

---

## Для \(n = (0,0,1)\):

\[
h_{ij} \sim T^{i3j3}.
\]

---

# 🔥 Ключевой результат №2

\[
\boxed{
h_{ij} = T^{i3j3}
}
\]

(с точностью до нормировки)

---

# 5. Проверка структуры

Подставляя E2:

\[
T^{ijkl} \sim
\delta^{ik}\delta^{jl} - \delta^{il}\delta^{jk}
\]

---

## получаем:

\[
h_{ij}
=
\delta^{i3}\delta^{j3} - \delta^{i3}\delta^{j3}
= 0
\]

---

⚠️ Это показывает:

простой изотропный E2 ещё не даёт TT-тензор —
нужна анизотропия (выбор оси + базис поляризаций).

---

# 6. Правильная процедура

Не напрямую из δ, а через разложение:

\[
T^{ijkl}
=
\alpha_+\,E^{(+)}_{ijkl}
+
\alpha_\times\,E^{(\times)}_{ijkl}.
\]

---

## где

\[
E^{(+)}_{ijkl}
\sim
e^{(+)}_{ij} e^{(+)}_{kl},
\]

\[
E^{(\times)}_{ijkl}
\sim
e^{(\times)}_{ij} e^{(\times)}_{kl}.
\]

---

# 7. Тогда

\[
\boxed{
h_{ij} =
\alpha_+\,e^{(+)}_{ij}
+
\alpha_\times\,e^{(\times)}_{ij}
}
\]

получается корректно.

---

# 8. Динамика

Из предыдущего:

\[
\Box \alpha_+ = 0,\quad \Box \alpha_\times = 0
\]

---

## ⇒

\[
\boxed{
\Box h_{ij} = 0
}
\]

в TT-калибровке.

---

# 🔥 Ключевой результат №3

\[
\boxed{
\text{линейные уравнения гравитационных волн возникают автоматически}
}
\]

---

# 9. Полная «метрика»

\[
\boxed{
g_{ij}(x) = \delta_{ij} + \epsilon\,h_{ij}(x)
}
\]

где \(\epsilon\) — малость возмущения.

---

# 10. Интерпретация

- \(\delta_{ij}\) — фон  
- \(h_{ij}\) — колебания (гравволны)  

---

## ⇒

\[
\boxed{
\text{метрика возникает как параметризация E2-мод}
}
\]

---

# 11. Неочевидная связь

Исходно:

- была только алгебра 4-тензора  
- без пространства-времени  

---

В результате:

\[
\boxed{
\text{геометрия (метрика) появляется как эффективная степень свободы}
}
\]

---

# 🔴 Итог

\[
\boxed{
h_{ij} \;\leftrightarrow\; (\alpha_+,\alpha_\times)
}
\]

\[
\boxed{
\text{две поляризации полностью определяют метрику в линейном приближении}
}
\]

---
