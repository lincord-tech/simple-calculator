### Идея 1: Решение в плоском фоне + оценка самосогласованности

**Шаг 1.1.** Решаем уравнение в плоском пространстве

Уравнение:  
**Δu = 0**, где u = e^{3Φ} = ρ, а Δ — обычный лапласиан в евклидовом 3D.

В сферических координатах для радиальной функции:

$$
\frac{1}{r^2} \frac{d}{dr} \left( r^2 \frac{du}{dr} \right) = 0
$$

Интегрируем дважды:

$$
r^2 \frac{du}{dr} = C \quad \Rightarrow \quad \frac{du}{dr} = \frac{C}{r^2} \quad \Rightarrow \quad u(r) = A - \frac{C}{r}
$$

Таким образом:

$$
\Phi(r) = \frac{1}{3} \ln \left( A - \frac{C}{r} \right)
$$

**Горизонт** определяется условием u(r_h) = 0:

$$
r_h = \frac{C}{A}
$$

**Шаг 1.2.** Основные свойства решения

- При r → ∞:  
  u(r) ≈ A, Φ(r) ≈ (1/3) ln A − (C/(3A r))  
  → Φ ∼ −1/r  (ньютоновский спад)

- При r → r_h⁺:  
  u(r) ≈ A (r − r_h)/r_h  
  Φ(r) ≈ (1/3) ln(r − r_h) + const  
  ∇Φ ≈ (1/(3(r − r_h))) ê_r  
  (∇Φ)² ≈ 1/(9 (r − r_h)²) — сильная сингулярность градиента

**Шаг 1.3.** Оценка backreaction (насколько решение самосогласованно)

Метрика, которую мы использовали, была плоской: h_{ij} = δ_{ij}.  
Однако в модели √h = e^{3Φ}, поэтому настоящая пространственная метрика должна быть  

h_{ij} ≈ e^{2Φ(r)} δ_{ij}   (конформная поправка)

Оценим величину поправки:

- Далеко от горизонта (r ≫ r_h): Φ мал, e^{2Φ} ≈ 1 + 2Φ ≈ 1 − (2C)/(3A r) — слабая поправка, решение почти самосогласованно.
- Близко к горизонту (r − r_h ≪ r_h): Φ → −∞, e^{2Φ} → 0 очень быстро.

Это значит, что **плоское приближение ломается** вблизи r_h. Метрика сильно сжимается (объёмный элемент √h → 0), что и соответствует смыслу горизонта в модели.

Вывод по Идее 1:

### Идея 2: Поток и температура на этом фоне

Теперь используем полученное решение и посчитаем **поток** и **эффективную температуру**.

**Шаг 2.1.** Плотность и поток

∇ρ = C / r² ê_r

Диффузионный поток:

J = −D ∇ρ = −D (C / r²) ê_r

Поток **конечен** и не равен нулю.

**Шаг 2.2.** Эффективная температура

Из модели у нас есть естественное определение температуры через градиент:

Около горизонта ρ ≈ (A / r_h) (r − r_h) = κ' (r − r_h), где κ' = A / r_h = A² / C

Тогда:

dt ≈ dτ / [κ' (r − r_h)]

Это приводит к логарифмическому redshift:

t ∼ (1/κ') ln(1/(r − r_h))

Таким образом, эффективная поверхностная гравитация:

**κ = A² / C = 1 / r_h**

Тогда температура:

**T = κ / (2π) = 1 / (2π r_h)**

(в единицах, где D = 1; в общем случае T ∼ D / r_h, но D = ε² — микроскопический параметр, который можно перенормировать).

**Шаг 2.3.** Масса и первый закон

Это точно совпадает со стандартной формулой Хокинга T = 1/(8π M) (с точностью до численного коэффициента, который можно подогнать выбором единиц).

### Точное выражение Laplace–Beltrami для радиальной функции

\[
\Delta_h u = \frac{1}{\sqrt{h}}\,\partial_r\Bigl(\sqrt{h}\,h^{rr}\,\partial_r u\Bigr)
\]

Подставляем \(\sqrt{h}\) и \(h^{rr}\):

\[
\Delta_h u = \frac{1}{r^2 e^{3\Phi}}\,\partial_r\Bigl(r^2 e^{3\Phi}\cdot e^{-2\Phi}\,\partial_r u\Bigr) = \frac{1}{r^2 e^{3\Phi}}\,\partial_r\Bigl(r^2 e^{\Phi}\,\partial_r u\Bigr)
\]

Теперь \(\partial_r u = 3\,e^{3\Phi}\,\Phi'\) (штрих = \(d/dr\)):

\[
\Delta_h u = \frac{1}{r^2 e^{3\Phi}}\,\partial_r\Bigl(r^2 e^{\Phi}\cdot 3 e^{3\Phi} \Phi'\Bigr) = \frac{3}{r^2 e^{3\Phi}}\,\partial_r\Bigl(r^2 e^{4\Phi}\,\Phi'\Bigr)
\]

Уравнение стационарности \(\Delta_h u = 0\) даёт:

\[
\partial_r\Bigl(r^2 e^{4\Phi}\,\Phi'\Bigr) = 0
\]

Интегрируем один раз:

\[
r^2 e^{4\Phi}\,\Phi' = K = \text{const}
\]

Получаем **точное ОДУ первого порядка**:

\[
\Phi' = \frac{K}{r^2 e^{4\Phi}}
\]

или, вводя \(w(r) = e^{4\Phi}\):

\[
w' = 4K\,\frac{w}{r^2} \qquad \Rightarrow \qquad \frac{dw}{w} = 4K\,\frac{dr}{r^2}
\]

Решение:

\[
\ln w = -\frac{4K}{r} + C_0 \qquad \Rightarrow \qquad w(r) = A\,e^{-4K/r}
\]

(где \(A = e^{C_0}\)). Возвращаемся к \(\Phi\):

\[

\]

Переобозначаем константы (\(B = \frac14\ln A\), \(C = K\)):

\[

- Асимптотика:

\]

### 1. Поток

Диффузионный поток:

\[
\mathbf{J} = -D \nabla \rho = -D \nabla u
\]

В сферической симметрии:

\[
\frac{du}{dr} = \frac{C}{r^2} \quad \Rightarrow \quad J_r = -D \frac{C}{r^2}
\]

На горизонте \(r \to r_h^+\):

\[
|J_h| = D \frac{C}{r_h^2} = D \frac{C}{(C/A)^2} = D \frac{A^2}{C}
\]

Поток конечен и направлен наружу.

### 2. Температура

Используем определение температуры через redshift внутреннего времени графа \(\tau\) и наблюдаемого времени \(t\):

\[
dt = \frac{d\tau}{\rho} = \frac{d\tau}{u(r)}
\]

Около горизонта:

\[
u(r) \approx \frac{A}{r_h} (r - r_h) = \kappa (r - r_h), \quad \kappa = \frac{A}{r_h} = \frac{A^2}{C}
\]

Тогда:

\[
dt \approx \frac{d\tau}{\kappa (r - r_h)}
\]

Это приводит к логарифмическому накоплению фазы:

\[
t \sim \frac{1}{\kappa} \ln \frac{1}{r - r_h}
\]

Эффективная поверхностная гравитация:

\[
\kappa = \frac{A^2}{C} = \frac{1}{r_h}
\]

Температура Хокинга (в естественных единицах):

\[
T = \frac{\kappa}{2\pi} = \frac{1}{2\pi r_h}
\]

### 3. Вывод Bogoliubov-коэффициентов и спектра

Берём моду с частотой \(\Omega\) относительно внутреннего времени \(\tau\):

\[
\psi(\tau) = e^{-i \Omega \tau}
\]

Переходим к наблюдаемому времени \(t\) через соотношение \(dt = d\tau / u(r)\). Около горизонта это даёт:

\[
\tau(t) \approx \tau_0 - \frac{1}{\kappa} e^{-\kappa t}
\]

(стандартная экспоненциальная привязка).

Тогда мода в наблюдаемом времени:

\[
\psi(t) \propto \exp\left( i \frac{\Omega}{\kappa} e^{-\kappa t} \right)
\]

Чтобы найти Bogoliubov-коэффициенты, делаем Фурье-разложение по наблюдаемым частотам \(\omega\):

\[
\alpha_\omega \sim \int_{-\infty}^{\infty} dt \, e^{i\omega t} \exp\left( i a e^{-\kappa t} \right), \quad a = \frac{\Omega}{\kappa}
\]

Делаем замену \(z = e^{-\kappa t}\), \(dt = -dz/(\kappa z)\):

\[
\alpha_\omega \propto \int_0^\infty dz \, z^{-1 - i\omega/\kappa} e^{i a z}
\]

Это интеграл, выражающийся через гамма-функцию:

\[
\int_0^\infty z^{\nu-1} e^{i a z} dz \propto \Gamma(\nu) \, a^{-\nu} \, e^{i \pi \nu / 2}, \quad \nu = -i \frac{\omega}{\kappa}
\]

В результате получаем:

\[
\frac{|\beta_\omega|^2}{|\alpha_\omega|^2} = e^{-2\pi \omega / \kappa}
\]

Число частиц в моде с частотой \(\omega\):

\[
n(\omega) = |\beta_\omega|^2 = \frac{1}{e^{2\pi \omega / \kappa} - 1}
\]

**Это точно тепловой спектр Бозе–Эйнштейна** с температурой

\[
T = \frac{\kappa}{2\pi} = \frac{1}{2\pi r_h}
\]

- **Поток** на горизонте: \(|J_h| = D A^2 / C > 0\) (конечный, наружу).
- **Температура**: \(T = 1/(2\pi r_h)\), scaling \(T \sim 1/M\) (если \(r_h \sim M\)).
- **Спектр**: Получен тепловой спектр \(n(\omega) = 1/(e^{\omega/T}-1)\) из комбинаторной динамики путей, без квантования поля на фоне.

Число частиц в моде с частотой \(\omega\):

\[
n(\omega) = |\beta_\omega|^2 = \frac{1}{e^{2\pi \omega / \kappa} - 1}
\]

**Это точно тепловой спектр Бозе–Эйнштейна** с температурой

\[
T = \frac{\kappa}{2\pi} = \frac{1}{2\pi r_h}
\]

### Итоговые результаты

- **Поток** на горизонте: \(|J_h| = D A^2 / C > 0\) (конечный, наружу).
- **Температура**: \(T = 1/(2\pi r_h)\), scaling \(T \sim 1/M\) (если \(r_h \sim M\)).
- **Спектр**: Получен тепловой спектр \(n(\omega) = 1/(e^{\omega/T}-1)\) из комбинаторной динамики путей, без квантования поля на фоне.
