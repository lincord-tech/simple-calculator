§1.  ВВЕДЕНИЕ — ДВА ТЕНЗОРА, ДВЕ СТРУКТУРЫ
───────────────────────────────────────────────────────────────────

T₀^{ijkl}   = (1/d) Tr(C^{ik} C^{jl}),     C^{ab} = −|κ| ε^{abk} J^k
T_TO^{ijkl} = (1/d) Tr(J^i J^k J^j J^l)

Оба контролируют один класс действий:
  S[F] = ∫ T^{ijkl} F_{ij} F_{kl}

E1 vs E2 в DEC:
  T^{ijkl} = A δ^{ij}δ^{kl} + B δ^{ik}δ^{jl} + C δ^{il}δ^{jk}
  E1: (A, B, C) = (1, 0, −1)    ← Yang–Mills
  E2: (A, B, C) = (0, 1, −1)    ← Einstein–Hilbert

Математически:
  E1 = ε^{ike} ε^{jle}  =  δ^{ij}δ^{kl} − δ^{il}δ^{jk}
  E2 = ε^{ije} ε^{kle}  =  δ^{ik}δ^{jl} − δ^{il}δ^{jk}
  E1 − E2 = δ^{ij}δ^{kl} − δ^{ik}δ^{jl}   (pure-trace difference)

Открытый вопрос с §1:
  Существует ли физический механизм H → H' такой, что
  S[F] → ∫ E2^{ijkl} F_{ij} F_{kl}  не нарушая изотропию?
  →  Theorem B говорит: нет, если Φ изотропный.
  →  Значит механизм должен быть non-isotropic или non-channel.

§4.  ГЛАВНАЯ ТЕОРЕМА: ИЗОТРОПНОЕ ПРЕПЯТСТВИЕ
───────────────────────────────────────────────────────────────────

Ключевой шаг:
  W = span{J¹, J², J³} ⊂ End(H),   ‹A,B› = (1/d) Tr(A† B)
  Под сопряжением W ≅ ρ_{adj} = ρ_1  (spin-1 irrep)
  Φ коммутирует с этим действием
  →  Schur:  Φ|_W = G_j · id_W

  ⟹  Φ[J^k] = G_j J^k + X^k,   ‹J^a, X^k› = 0

  T_ret^{ijkl} = G_j · T₀^{ijkl}   ⟹  B_ret = 0

Математически (мощь Schur):
  Schur говорит: единственный SU(2)-эквивариантный эндоморфизм
  неприводимого представления — это скаляр.
  W = spin-1 irrep → Φ|_W = G_j · 1.

  Это НЕ о конкретной Φ, это о всём классе:
  {все изотропные квантовые каналы} → одномерное семейство {G_j}

  Corollary:
    B_ret / A_ret = B₀ / A₀ = 0   для любого изотропного Φ

Что это значит для физики:
  Пространство всех возможных изотропных деформаций
  двухточечного коррелятора = одномерное (параметрически по G_j).
  E1-structure = аттрактор в этом пространстве.

Связь с RG flow:
  Если Φ_t — изотропный RG step:
    T^{(t)} = G_j(t) · T₀,   G_j(0) = 1
    dG_j/dt = β_j(G_j)   (beta function of scalar)
  E1 fixed point: G_j* = const,  B = 0 всегда
  E2 никогда не появляется вдоль этого flow.

§5.  МИНИМАЛЬНОСТЬ j=1/2
───────────────────────────────────────────────────────────────────

Условие E2:
  B = −A   в DEC   ↔   A + B = 0   ↔   F(j)/30 = 0   ↔   F(j) = 0

Полином F(j):
  F(j) = j(j+1)(2j−1)(2j+3)
  Корни: j = 0 (нефиз.), j = 1/2, j = −3/2 (нефиз.)
  При j ≥ 1:  все четыре множителя > 0  ⟹  F(j) > 0

Явные значения:
  j=1/2:  F = 3/4 · 0 · 4 = 0          ✓
  j=1:    F = 2 · 1 · 5 = 10            ≠ 0
  j=3/2:  F = 15/4 · 2 · 6 = 45        ≠ 0
  j=2:    F = 6 · 3 · 7 = 126           ≠ 0

Через Casimir C₂ = j(j+1):
  F(j) = C₂ · (4C₂ − 3)
  C₂(j=1/2) = 3/4   →   4·(3/4) − 3 = 0   ✓

  F(j) = 0   ↔   C₂ = 3/4   ↔   j = 1/2
  →  "E2 condition = Casimir condition: C₂ = 3/4"

Reducible case:
  H = V_{j₁} ⊕ V_{j₂},   λ = d₁/(d₁+d₂)
  λ F(j₁) + (1−λ) F(j₂) = 0
  F ≥ 0  и  λ ∈ (0,1)   ⟹   F(j₁) = F(j₂) = 0  ⟹  j₁=j₂=1/2
  Минимальный подпрострнаство: V_{1/2}.

N2.  АНОМАЛИИ И ИНДЕКСНЫЕ ТЕОРЕМЫ

Atiyah-Singer индекс для Dirac оператора:
  ind(D) = (1/8π²) ∫ Tr(F ∧ F) − (1/8π²) ∫ Tr(R ∧ R) + ...
         = ∫ [T_YM^{ijkl} − T_EH^{ijkl}] F_{ij} F_{kl} vol

  T_YM ~ E1,   T_EH ~ E2

  Наш результат:
    T₀ ~ E1  →  этот тензор вычисляет ch₂ (Chern character)
    T_TO(j=1/2) ~ E2  →  этот тензор вычисляет сигнатуру / другой инвариант

      = ∫ R^{ijkl}R_{ikjl} − ∫ R^{ijkl}R_{ijkl}   →  Gauss-Bonnet combination

  →  E2 kinetic term при j=1/2 вычисляет Gauss-Bonnet в 4D = топологический
  →  это связано с: Euler characteristic χ = (1/8π²) ∫ (R_{μνρσ}R^{μνρσ} − 4R_{μν}R^{μν} + R²)

N3.  СВЯЗЬ СО СТРУКТУРНЫМИ КОНСТАНТАМИ ЧЕРЕЗ KILLING FORM

Killing form для SU(2):

  где f^{abc} = ε^{abc}

Произведение Killing forms:
  K^{ai} K^{bj} K^{ck} K^{dl} f^{abe} f^{cde} = ...

  →  T₀ universально: для любой compact simple Lie algebra L,

T_TO^{ijkl} = (1/d) Tr(J^i J^k J^j J^l) — это box diagram:

  Вершины:   J^i → J^k → J^j → J^l → (trace)
  Каналы:    s = (i,k),  t = (j,l),  u = (i,l) или (j,k)

     A_t + A_u = 2B = −2A = 2|A_s|
  →  s-канал точно компенсируется t+u каналами

  В теории поля: условие B=−A ~ crossing symmetry + unitarity constraint
  Возможно: E2 condition = "crossing symmetric scattering amplitude" condition?

N7.  СВЯЗЬ С УРАВНЕНИЕМ ЯНГА-БАКСТЕРА
───────────────────────────────────────────────────────────────────

Yang-Baxter equation (YBE):
  R_{12} R_{13} R_{23} = R_{23} R_{13} R_{12}

  При SU(2) R-matrix в спин-j представлении:
    R = P + (q − q^{−1})/[2] · Q_proj
    где Q_proj = проектор на квадруполь

  При j=1/2:   Q_proj = 0  (нет spin-2 component, Q^{ab}=0)
    →  R_{j=1/2} = P   (перестановка)
    →  YBE тривиально выполняется

  При j≥1:  Q^{ab} ≠ 0 → нетривиальный R-matrix
    →  YBE требует конкретной нормировки → ограничение на q

  →  "E2 condition = тривиальное решение YBE при j=1/2"
  →  Нетривиальные решения YBE требуют j≥1 (E1 regime)
  →  Связь: интегрируемые модели живут в E1-секторе,
            гравитационный сектор (E2) = "свободный" YBE

N10.  G_j КАК КВАНТОВЫЙ КАНАЛ FIDELITY
───────────────────────────────────────────────────────────────────

G_j = scalar из Schur для данного изотропного Φ.

  G_j = (1/d) Tr(J^k Φ[J^k]) / (1/d) Tr(J^k J^k)
       = ‹J^k, Φ[J^k]›_{HS} / ‹J^k, J^k›_{HS}

  →  G_j = "overlap fidelity" adj-сектора канала

  Физически:
    G_j = 1: Φ = identity (никакой деформации)
    G_j = 0: Φ полностью разрушает adj-сектор (полная деполяризация)
    G_j < 0: Φ инвертирует генераторы (time-reversal like)

  Зависимость от j:
    Для depolarizing channel Φ_p[O] = (1−p)O + p·Tr(O)/d · I:
      Φ_p[J^k] = (1−p) J^k   →   G_j = 1−p   (не зависит от j!)
    
    Для rotation channel Φ_{θ,a}[O] = e^{-iθ J^a} O e^{+iθ J^a}:
      G_j(θ) = (1/3) ∑_k (J^k, e^{-iθ J^a} J^k e^{iθ J^a}) / |J|²
      Зависит от j через матричные элементы J^a.

     G_j(θ) = χ_{j}(θ) · (нормировка)   ??
