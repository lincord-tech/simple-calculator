Discrete Transition Geometry Model (σ-model)

0. Purpose

This document is a complete, self-contained description of a discrete model where:

space is not fundamental

time is not fundamental

both emerge from a single microscopic transition process

The goal is not to assume geometry, but to generate it from dynamics.

1. Primitive structure (what exists fundamentally)

There is no background space.

The only primitive ingredients are:

1.1 Nodes (events / sites)

A set of elements:

x_i

These are not points in space. They have no coordinates.

1.2 Connectivity (allowed transitions)

A relation between nodes:

x_i ~ x_j

Meaning:

A transition can occur between i and j in one elementary step.

This defines a graph.

1.3 Internal state (duality)

Each node carries a binary internal state:

σ_i ∈ {−1, +1}

Interpretation:

fundamental dual degree of freedom

not spatial, not directional yet

This is the only internal variable.

2. Fundamental process (transition)

2.1 Non-instantaneous switching

A state change cannot occur instantly.

It must pass through an intermediate state:

+1 → 0 → −1

The state 0 is NOT an independent persistent state.

It is:

a transition event

a temporal midpoint

2.2 Elementary transition time

Each crossing through 0 takes a fixed time:

τ

This defines the minimal temporal unit.

3. Emergent quantities

3.1 Time

Time is defined as the count of transitions:

T = N_0

where N_0 is the number of times σ passes through 0.

Time is therefore:

discrete

event-based

not a background parameter

3.2 Distance

Distance is defined via connectivity:

D(i,j) = a × (minimal number of transitions between i and j)

where a is the elementary step.

3.3 Fundamental relation

The model introduces one physical scale:

c = a / τ

This is the maximum propagation speed.

It is not assumed independently.

4. Interpretation

We do not have:

particles in space

We have:

transitions between states

Thus:

space = structure of possible transitions

time = count of transitions

5. Local dynamics

We introduce a local interaction rule:

S = Σ⟨i,j⟩ (σ_i − σ_j)^2

This favors alignment of neighboring states.

Constraint: Direct transition +1 ↔ −1 is forbidden. It must pass through 0.

6. Wave emergence

Consider a local perturbation:

σ_i = +1 in a region

Neighbors attempt to minimize mismatch.

But:

transitions require time τ

propagation is stepwise

Result:

A disturbance propagates.

This is a wave.

7. Propagation speed

Because one step takes τ and covers a:

v_max = a / τ = c

Thus:

no signal exceeds c

8. Emergence of dimensionality

Dimension is not assumed.

It is defined via growth of reachable nodes:

N(r) ~ r^d

Thus:

d = lim (log N(r) / log r)

If connectivity ~6 neighbors → d ≈ 3

9. Geometry from statistics

Define correlation:

C(i,j) = ⟨σ_i σ_j⟩

Effective metric:

g^{ij} ~ C(i,j)

Thus geometry emerges from correlations.

10. Gravity (hypothesis)

Define local transition density:

ρ_0(x) = frequency of σ=0 events

Interpretation:

√g_00(x) ~ ρ_0(x)

Thus:

varying transition rates → varying time flow

this mimics gravitational potential
