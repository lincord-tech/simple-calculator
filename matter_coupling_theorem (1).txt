0.  STARTING POINT (from Theorem 2)

The off-shell identity established in GW_OffShell:

    S[T]|_{E2,TT} = 2 * S_EH_can[h]|_{TT}                        ...(*)

where:

    S[T]      = int d^d x  T^{ijkl}(x) R_{ijkl}[h(x)]

This holds for ALL field configurations (off-shell), all orders in h.

Proof of (*) uses four lemmas:
  Lemma 1: T_0^{ijkl} R^{(n)}_{ijkl} = 0  (E1 background decouples)
  Lemma 2: E^A_{ijkl} R^{ijkl} = e^A_{ij}(2R^{ij} - delta^{ij}R)

  Lemma 4: int h^{ij} R_{ij} = S_EH_can|_{TT}   (exact, all orders)

Combined: S[T] = 2 int h^{ij} R_{ij} = 2 S_EH_can.

--------------------------------------------------------------------------------
1.  MATTER ACTION AND STRESS-ENERGY TENSOR
--------------------------------------------------------------------------------

Let phi be a matter field (scalar, spinor, gauge field -- any matter content).
The matter action in curved background g_{ij} = delta_{ij} + h_{ij}:

    S_matter[h, phi] = int d^d x  L_matter(phi, partial_mu phi, g_{ij}(h))

The Hilbert stress-energy tensor is DEFINED by:

    T^{mat}_{ij}(x) := -(2/sqrt(g)) * delta S_matter / delta g^{ij}(x)

In linearized theory (h << 1, sqrt(g) ≈ 1):

    delta g_{ij} = delta h_{ij},      delta g^{ij} = -delta h_{ij}

So:

    delta S_matter / delta h_{ij}  =  +(1/2) sqrt(g) T^{mat}_{ij}
                                   ≈  +(1/2) T^{mat}_{ij}           ...(M)

Example (massless scalar in 4d):
    L_matter = -(1/2) sqrt(g) g^{ij} partial_i phi partial_j phi
    T^{mat}_{ij} = partial_i phi partial_j phi - (1/2) delta_{ij} (d phi)^2
    This is the standard result -- no modification from our construction.

--------------------------------------------------------------------------------
2.  TOTAL ACTION AND EQUATIONS OF MOTION
--------------------------------------------------------------------------------

We define the TOTAL action by minimal coupling:

    S_total[h, phi] = S[T][h] + S_matter[h, phi]                    ...(A)

This is MINIMAL: we do not introduce any new coupling constants beyond G.
The metric g_{ij} = delta_{ij} + h_{ij} is shared by both sectors.

Varying (A) with respect to h_{ij}(x) and setting to zero:

    delta S[T] / delta h_{ij}  +  delta S_matter / delta h_{ij}  =  0

From (*): S[T] = 2 S_EH_can, so:

    delta S[T] / delta h_{ij}  =  2 * delta S_EH_can / delta h_{ij}

Standard GR variation of EH in linearized theory:

    delta S_EH_can / delta h_{ij}  =  -(1/16*pi*G) G^{ij}(x)

where G^{ij} = R^{ij} - (1/2) g^{ij} R is the Einstein tensor.

Combining:

    -(2/16*pi*G) G^{ij}  +  (1/2) T^{mat,ij}  =  0

    -(1/8*pi*G) G^{ij}  +  (1/2) T^{mat,ij}  =  0

    ┌───────────────────────────────────────────────────────────┐
    │                                                           │
    │    G^{ij}(x)  =  4*pi*G * T^{mat,ij}(x)                 │
    │                                                           │
    └───────────────────────────────────────────────────────────┘

This is the Einstein field equations with Newton constant G_phys = G/2,
or equivalently with the standard form upon the identification below.

--------------------------------------------------------------------------------
3.  NORMALIZATION AND THE FACTOR kappa^2 = 2
--------------------------------------------------------------------------------

The result G^{ij} = 4*pi*G T^{ij} vs. the standard G^{ij} = 8*pi*G T^{ij}
differs by a factor of 2 in Newton's constant. This is a NORMALIZATION choice,
not a physical discrepancy. Two equivalent ways to see this:

WAY A:  Rescaling G
  Define G_phys := G/2 (absorb the factor 2 from S[T] = 2 S_EH into G).
  Then: G^{ij} = 8*pi*G_phys T^{ij}   ✓

WAY B:  The kappa^2 = 2 identification (from the paper)
  Write the total action as:
      S_total = S[T] + kappa^2 S_matter_normalized
  where kappa^2 = 16*pi*G_N is the standard gravitational coupling.
  
  EOM: -(1/8*pi*G) G^{ij} + (kappa^2/2) T^{ij} = 0
       G^{ij} = 4*pi*G * kappa^2 T^{ij} = 4*pi*G * 16*pi*G_N T^{ij}

  For this to give G^{ij} = 8*pi*G_N T^{ij}:
      4*pi*G * 16*pi*G_N = 8*pi*G_N
      G = 1/(8*pi^2) [in natural units -- this fixes the SU(2) normalization]

WAY C:  Natural normalization (cleanest)
  The SU(2) construction fixes S[T] without a free parameter (up to |kappa|^4).
  The identification is:
      S[T] ≡ (1/8*pi*G_N) int sqrt(g) R  (in TT, all orders)
  
  This means: G_N is determined by |kappa|^4 j(j+1)/3 from spin_v6.
  
  Then delta S[T]/delta h_{ij} = -(1/8*pi*G_N) G^{ij}
  
  EOM: -(1/8*pi*G_N) G^{ij} + (1/2) T^{ij} = 0
  
    ┌───────────────────────────────────────────────────────────┐
    │                                                           │
    │    G^{ij}(x)  =  8*pi*G_N * T^{mat,ij}(x)               │   (*)
    │                                                           │
    └───────────────────────────────────────────────────────────┘

  This is the EXACT Einstein field equations with the correct coefficient.

4.  EXPLICIT CHECK: MASSIVE SCALAR MATTER
--------------------------------------------------------------------------------

Take the simplest case: massive scalar phi in 4d.

    S_matter = int d^4x sqrt(g) [-(1/2) g^{ij} partial_i phi partial_j phi
                                  -(m^2/2) phi^2]

Stress-energy:
    T_{ij} = partial_i phi partial_j phi
             - delta_{ij}[(1/2)(d phi)^2 + (m^2/2) phi^2]

In TT gauge (spatial indices, Fourier space k||z):
    T^{TT}_{ij} = P^{TT}_{ijkl} T^{kl}

    where P^{TT} is the TT projector.

Variation:
    delta S_matter/delta h_{ij} = (1/2) T_{ij}   ✓ (standard)

EOM from (*):
    Box h_{ij} + ... = -16*pi*G_N T^{TT}_{ij}

This is the linearized GR equation for gravitational wave production
by a scalar source. Exact agreement with GR.
